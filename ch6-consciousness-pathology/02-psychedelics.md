# 🧠 환각제 — 5-HT2A 와 의식의 재구성

*고전 환각제는 의식을 *끄는* 것이 아니라, 5-HT2A 수용체를 작용해 피질 흥분성과 신경 신호의 다양성(엔트로피)을 높이고 지각의 사전 확률을 느슨하게 풀어, 세계가 *다르게* 구성되도록 의식의 내용을 재편한다.*

## 🎯 핵심 질문

앞 문서에서 마취는 통합을 *끊어* 의식을 껐다. 환각제는 정반대 방향을 보여준다. LSD나 실로시빈을 복용한 사람은 의식을 *잃지* 않는다 — 오히려 깨어 있고, 종종 *더 깨어 있다고* 느낀다. 그러나 그가 경험하는 세계는 근본적으로 달라진다. 벽이 숨 쉬고, 음악에 색이 보이고, 자아의 경계가 녹아 세계와 하나가 되는 듯한 느낌(자아 해체, ego dissolution)이 찾아온다. 무엇이 이런 *재구성* 을 일으키는가?

소박한 답은 "환각제가 지각을 *강화* 하거나 *증폭* 한다 — 모든 것이 더 강렬해진다" 이다. 그러나 이 답은 핵심을 놓친다. 환각제 경험의 본질은 단순한 *볼륨 증가* 가 아니라 지각의 *조직 원리* 자체의 변화다. 익숙한 사물이 낯설어지고, 의미가 과잉으로 흘러넘치고, 평소라면 *무시* 되었을 미세한 패턴이 전경으로 솟아오른다. 이것은 신호를 키운 것이 아니라, 무엇을 신호로 *간주할지* 의 규칙을 바꾼 것이다.

이 문서의 핵심 질문은 이것이다. **5-HT2A 수용체의 작용은 피질의 어떤 계산 속성 — 흥분성, 신호 다양성, 사전 확률의 무게 — 을 바꿔 지각을 재구성하는가?** 그리고 그 변화를 "엔트로피 증가" 나 "사전 확률 이완" 으로 정량화한다 해도, *왜 그 상태가 그런 특정한 질감* — 경외, 의미의 범람, 자아의 용해 — *으로 느껴지는가* 는 어디까지 남는가? Carhart-Harris 등의 "Entropic Brain"(2014)과 REBUS 모델(Carhart-Harris & Friston, 2019)을 축으로 본다.

## 🌍 어디서 마주치나

환각제의 효과는 실험실 안에만 있지 않다 — 인류 문화와 임상 양쪽에 깊이 새겨져 있다.

- **자아 해체(ego dissolution)**: 고용량에서 흔히 보고되는, 자아와 세계의 경계가 녹는 경험. "내가 사라지고 모든 것과 하나가 되었다" 는 보고는 디폴트 모드 네트워크(DMN)의 비동기화와 상관한다.
- **지각의 유동화**: 정적인 사물이 움직이고, 패턴이 살아나고, 감각이 교차한다(공감각). 평소 안정적이던 지각이 *흔들린다*.
- **의미의 범람**: 평범한 대상이 압도적 의미를 띠고, 우연한 연결이 깊은 통찰로 다가온다 — "현저성(salience)" 의 과잉 할당.
- **치료적 맥락**: 실로시빈 보조 치료가 난치성 우울·말기 환자의 실존적 고통·중독에서 임상 시험 중이다 — "경직된" 사고 패턴을 *느슨하게* 푸는 효과가 가설이다.
- **음악·예술과의 결합**: 환각 경험에서 음악이 감정·심상을 강력하게 끌어내는 것은, 사전 확률이 풀린 뇌가 외부 구조에 더 *열려* 있기 때문이라는 해석.

## 🔍 직관의 함정

**함정 1 — "환각제는 지각을 강화한다 — 모든 것을 더 강렬하고 선명하게 만든다."**
직관은 환각을 *지각의 볼륨 손잡이를 끝까지 올린 것* 으로 본다. 반증: 만약 단순 증폭이라면 효과는 *균일* 해야 하지만, 실제 환각 경험은 선택적이고 *왜곡* 적이다 — 어떤 것은 솟아오르고 어떤 것은 녹아내리며, 안정적이던 형태가 *불안정* 해진다. 더 결정적으로, 환각제는 일차 감각의 *해상도* 를 높이지 않는다(시력 검사표를 더 잘 읽게 되지 않는다). 바뀐 것은 입력의 강도가 아니라, 입력을 *해석하는* 상위 모델 — 무엇을 기대하고 무엇을 신호로 칠지 — 이다. 환각은 증폭이 아니라 *재구성* 이다.

**함정 2 — "환각은 그냥 무작위 잡음이다 — 뇌가 고장 나 헛것을 본다."**
반대 극단의 직관은 환각을 *순수한 노이즈* 로 본다. 반증: 환각 경험은 무작위가 아니라 *구조화* 되어 있다 — 기하학적 패턴(터널·격자·나선)은 시각 피질의 *해부학적 조직* 을 반영하고, 환각의 내용은 개인의 기억·정서·맥락에 깊이 의존한다. 정보이론적으로 보면 환각제 하의 뇌 신호는 *무작위(최대 엔트로피)* 가 아니라, 깨어 있는 정상 상태보다 *다양성(엔트로피)이 높지만 여전히 구조를 가진* 상태다. 잡음이 늘어난 것이 아니라, *접근 가능한 상태의 레퍼토리* 가 넓어진 것이다.

**함정 3 — "5-HT2A 작용은 세로토닌을 '더 많이' 만드는 것이다 — SSRI처럼."**
환각제도 SSRI도 세로토닌계를 건드리니 비슷하리라는 직관. 반증: SSRI는 세로토닌 *재흡수* 를 막아 시냅스 세로토닌을 전반적으로 늘리고 여러 수용체에 간접적으로 작용하며 효과가 *수주* 걸린다(Ch3-03). 환각제는 정반대로 *특정 수용체*(5-HT2A)에 직접 결합하는 *작용제(agonist)* 로, 세로토닌 양을 늘리는 것이 아니라 한 수용체 아형을 *직접 켜며*, 효과가 *수십 분 내* 극적으로 나타난다. 결정적 증거: 5-HT2A 길항제(케탄세린)를 미리 투여하면 LSD의 주관적 효과가 *대부분 차단* 된다 — 환각의 문은 5-HT2A 하나에 달려 있고, 이는 세로토닌 총량의 문제가 아니다.

## ⚙️ 회로·분자 분석

**분석 1 — 5-HT2A: 5층 추체 뉴런 위의 흥분성 스위치.**
고전 환각제(LSD·실로시빈의 활성대사물 실로신·DMT·메스칼린)의 공통 표적은 세로토닌 2A 수용체(5-HT2A)다. 이 수용체는 $G_q$ 단백질 연결형으로, 활성화되면 포스포리파제 C 경로를 통해 세포를 *탈분극* 시킨다 — 즉 흥분성이다. 결정적으로 5-HT2A는 피질 *5층 추체 뉴런(layer-5 pyramidal neurons)* 에 밀집해 있다. 5층 추체 세포는 피질의 주요 *출력* 뉴런이자 피질 영역들을 잇는 장거리 연결의 원천이다. 따라서 5-HT2A 작용은 피질의 핵심 출력·연결 노드를 *과흥분* 시킨다. 케탄세린(5-HT2A 길항제)으로 이 수용체를 막으면 환각 효과가 사라진다는 사실이 표적의 특이성을 못 박는다. $\square$

**분석 2 — 피질 흥분성 증가와 신호 다양성(엔트로피)의 상승.**
5층 추체 세포의 과흥분은 피질 활동의 통계적 성격을 바꾼다. 정상 각성 상태의 피질은 비교적 *제한된* 활동 패턴 레퍼토리 안에서 움직이지만, 환각제 하에서는 뉴런 집단이 더 *다양한, 덜 예측 가능한* 상태들을 방문한다. 이를 정보이론으로 재면, 신경 신호의 *엔트로피*(또는 신호 다양성·Lempel-Ziv 복잡도)가 *증가* 한다. Carhart-Harris 등(2014)의 "entropic brain" 가설은 바로 이것이다 — 환각 상태는 정상 각성보다 *더 높은 엔트로피* 의 상태이며, 그 엔트로피가 의식의 풍부함·유동성과 상관한다. MEG/fMRI에서 신호 다양성의 증가가 주관적 강도와 함께 가는 것이 관측되었다(Schartner 등 2017; Carhart-Harris 등 2016). $\square$

**분석 3 — 사전 확률의 이완: REBUS 모델.**
왜 엔트로피 증가가 지각의 *재구성* 으로 이어지는가? 예측 처리(predictive processing) 틀에서 뇌는 끊임없이 감각 입력을 *예측* 하고, 예측과 입력의 차이(예측오차)만 위로 전달한다. 이때 위계의 상위 *사전 확률(prior)* 은 예측오차에 *얼마나 무게를 줄지* — 그 정밀도(precision) — 를 통제한다. Carhart-Harris & Friston(2019)의 **REBUS**(RElaxed Beliefs Under pSychedelics) 모델은, 5-HT2A 작용이 상위 사전 확률의 *정밀도를 낮춘다* 고 본다. 사전이 느슨해지면 그것이 평소 *억눌러* 두던 하위 예측오차들이 위로 *흘러넘친다* — 그래서 평소 무시되던 패턴이 솟아오르고, 익숙한 해석이 풀리고, 세계가 *덜 정해진* 채로 다시 구성된다. 엔트로피 증가는 곧 사전의 이완이 만든 *상태 레퍼토리의 확장* 이다. $\square$

**분석 4 — 디폴트 모드 네트워크(DMN)의 비동기화와 자아 해체.**
환각제는 특히 디폴트 모드 네트워크 — 후측 대상피질·내측 전전두엽을 중심으로, 자기참조적 사고·자아감과 연관된 망 — 의 내부 통합을 *낮추고*, 평소 분리되어 있던 네트워크들 사이의 결합을 *높인다*. DMN은 위계의 *최상위 사전*(가장 추상적이고 안정적인 자기 모델)을 담는 곳으로 해석되는데, 그 비동기화는 REBUS에서 가장 높은 수준의 사전 — *자아라는 모델* — 이 풀리는 것에 대응한다. 그래서 고용량에서 자아의 경계가 녹는 *자아 해체* 가 나타난다. 회로 수준에서는 한 망의 결속이 풀리고 망 사이 경계가 흐려지는 *재배선* 이고, 경험 수준에서는 "나" 와 "세계" 의 구분이 사라지는 사건이다. $\square$

## 🧪 실험·임상 증거

### 실험 1 — 5-HT2A 게인에 따른 신호 다양성(Lempel-Ziv 엔트로피)의 증가

환각제를 "피질 활동의 다양성(엔트로피)을 높이는 5-HT2A 게인" 으로 모형화한다. 피질 신호를 다수 진동 모드의 혼합으로 두고, "게인" 파라미터를 올리면 활동이 더 많은 모드와 더 큰 비선형 요동을 방문하도록 한다. 신호를 이진화해 **Lempel-Ziv 복잡도**(시계열의 압축 불가능성 = 다양성의 대리지표)를 측정하면, 게인과 함께 복잡도가 *증가* 함을 본다. 이는 Carhart-Harris 등(2014)의 entropic brain 가설, Schartner 등(2017, *Sci Rep*)이 LSD·실로시빈·케타민에서 관측한 신호 다양성 증가를 단순화해 재현한다.

```python
import numpy as np

def _lz76(s):
    """이진 시퀀스의 Lempel-Ziv(LZ76) 복잡도 — 새로운 부분문자열의 개수.
       다양성/엔트로피가 높을수록 압축이 어려워 복잡도가 크다."""
    n = len(s)
    i, c, l, k = 0, 1, 1, 1
    k_max = 1
    while l + k <= n:
        if s[i + k - 1] == s[l + k - 1]:
            k += 1
        else:
            if k > k_max:
                k_max = k
            i += 1
            if i == l:
                c += 1
                l += k_max
                i = 0
                k_max = 1
            k = 1
    if k != 1:
        c += 1
    return c

def cortical_signal(gain, T=2000, n_modes=12, seed=0):
    """피질 신호 모형: 다수 진동 모드의 혼합 + 게인에 따라 커지는 비선형 요동.
       gain↑ = 5-HT2A 작용 = 더 많은 모드와 큰 요동 → 더 다양한 활동."""
    rng = np.random.default_rng(seed)
    t = np.arange(T)
    sig = np.zeros(T)
    for m in range(1, n_modes + 1):
        amp = 1.0 / m                                  # 1/f형 스펙트럼
        phase = rng.uniform(0, 2*np.pi)
        # 게인이 클수록 고차 모드의 기여와 위상 요동이 커진다
        weight = amp * (1 + gain * (m / n_modes))
        sig += weight * np.sin(2*np.pi*m*t/T * 8 + phase + gain*0.5*rng.standard_normal())
    sig += gain * 0.4 * rng.standard_normal(T)         # 게인에 비례하는 잡음 요동
    return sig

def normalized_lz(gain, seed=0):
    sig = cortical_signal(gain, seed=seed)
    binary = (sig > np.median(sig)).astype(int)        # 중앙값 기준 이진화
    lz = _lz76(''.join(map(str, binary)))
    n = len(binary)
    norm = n / np.log2(n)                              # 길이 정규화
    return lz / norm

print("[5-HT2A 게인 증가 → 신호 다양성(정규화 LZ 복잡도) 증가]")
print(f"{'게인':>6} {'정규화 LZ복잡도':>16}")
for g in (0.0, 0.5, 1.0, 1.5, 2.0):
    vals = [normalized_lz(g, seed=s) for s in range(8)]   # 8회 평균
    state = "정상 각성" if g == 0.0 else "환각 상태"
    print(f"{g:>6.1f} {np.mean(vals):>16.3f}   <- {state}")
# → 게인(5-HT2A 작용)이 커질수록 LZ 복잡도가 상승:
#   '엔트로피 증가 = 의식 내용의 다양화'라는 entropic-brain 가설을 재현한다.
```

실행하면 게인(5-HT2A 작용)이 0(정상 각성)에서 2.0(고용량 환각)으로 커질 때 정규화 LZ 복잡도가 *단조 증가* 한다 — 신호가 더 다양하고 덜 예측 가능한 상태들을 방문하기 때문이다. 이것이 entropic brain 가설의 핵심 주장의 회로적 그림이다: 환각제는 의식을 끄는(엔트로피를 낮추는) 마취의 정반대 — 의식의 *상태 레퍼토리를 넓혀* 엔트로피를 높이는 — 방향에 있다. 이 결과는 Schartner 등(2017)이 MEG에서 LSD·실로시빈·아산화질소·케타민 모두에서 신호 다양성 증가를 관측한 것, Carhart-Harris 등(2014, *Front Hum Neurosci*)의 entropic brain 가설과 정성적으로 일치한다.

**반례 노트**: 이 모델은 엔트로피(LZ 복잡도)를 "게인" 이라는 단일 파라미터의 증가로 만들었지만, 이는 *상관* 의 재현이지 *인과* 의 증명이 아니다 — 단순히 잡음을 늘려도 LZ는 오르므로, 진짜 환각의 엔트로피 증가가 *구조화된 레퍼토리 확장* 인지 *무작위 잡음 증가* 인지를 LZ 하나로는 구별하지 못한다(함정 2 참고). 또 신호 다양성의 증가가 *의식의 풍부함* 과 상관한다는 entropic brain의 핵심 주장은, 케타민(NMDA 차단·해리성)에서도 다양성이 오른다는 사실 때문에 복잡하다 — 엔트로피 증가만으로는 환각제의 *특이한* 현상학(자아 해체·의미 범람)을 5-HT2A 작용의 그것과 구별하지 못한다. 마지막으로, REBUS의 "사전 이완" 은 이 모델에 명시적으로 들어 있지 않다 — 게인은 엔트로피를 낳지만, 그 엔트로피가 *왜* 정밀도 하락(사전 이완)에서 오는지는 별도의 위계적 모델이 필요하다.

## 🌉 설명적 간극

✅ **회로가 설명하는 것**: 왜 환각제 효과가 5-HT2A 하나에 달려 있는가(케탄세린 차단), 왜 효과가 즉각적이고 극적인가(직접 작용제, 5층 추체 과흥분), 왜 지각이 *유동화* 되는가(사전 이완으로 예측오차 범람), 왜 고용량에서 자아가 해체되는가(DMN 비동기화 = 최상위 사전의 이완).

✅ **계산이 환원하는 것**: 환각 상태를 정상 각성보다 *높은 엔트로피*(신호 다양성·LZ 복잡도)의 상태로, 지각 재구성을 사전 확률 정밀도의 *하락*(REBUS)으로, 자아 해체를 위계 최상위 사전의 이완으로 정량화·형식화할 수 있다는 것.

❌ **남는 것**: 엔트로피가 *왜* 그런 특정한 질감으로 *느껴지는가*? 모델은 신호 다양성의 *양* — LZ 복잡도의 수치 — 을 그리지만, 그 다양성이 *경외* 로, *의미의 범람* 으로, *경계의 용해* 로 경험되는 그 *질적 색깔* 은 그리지 못한다. 엔트로피 2.0이라는 숫자와 "모든 것이 살아 있고 의미로 가득 찼다" 는 1인칭 사이에는, 어떤 정보 측정으로도 건널 수 없는 간극이 있다.

❌ **또 남는 것**: 사전 확률이 풀린 뇌가 *왜* 흔히 깊은 *정서적·실존적 의미* 의 경험(우주와의 합일, 죽음에 대한 화해)을 동반하는가? REBUS는 사전이 풀려 예측오차가 흘러넘친다는 *구조* 를 그리지만, 그 풀림이 *왜* 단순한 혼란이 아니라 종종 *심오함* 으로 느껴지는지 — 그 정서적 깊이가 어디서 오는지 — 는 정밀도 파라미터의 하락으로 환원되지 않는다.

## 🧬 횡단 원리

**예측 처리에서 사전과 입력의 균형** 은 지각 전반을 지배하는 횡단 원리다. 정상 지각은 사전(기대)과 감각 입력(증거)의 *베이즈적 절충* 이다 — 사전이 너무 강하면 입력을 무시한 환각·착각이, 너무 약하면 입력에 휘둘리는 불안정이 온다. 환각제는 이 다이얼의 *사전 쪽 무게를 낮추는* 개입이고, 그 반대편에는 사전이 *과도하게 경직된* 상태들 — 강박적 반추, 우울의 부정적 자기 모델, 중독의 고착된 보상 예측 — 이 있다. 그래서 환각제의 치료 가설은 "경직된 사전을 일시적으로 풀어 재학습의 창을 연다" 는 것이다 — 같은 다이얼의 반대 방향 조정.

여기서 더 깊은 횡단 원리가 드러난다: **엔트로피와 의식의 풍부함**. 마취(Ch6-01)는 통합을 끊어 엔트로피를 *낮추고* 의식을 껐고, 환각제는 사전을 풀어 엔트로피를 *높여* 의식의 내용을 풍부하게 했다. 이 둘을 한 축에 놓으면 — 깊은 마취(최저 엔트로피·무의식) → 수면 → 정상 각성 → 환각 상태(최고 엔트로피) — 의식의 *내용 다양성* 이 엔트로피와 함께 변하는 스펙트럼이 보인다. entropic brain 가설의 야심은 이것이다: 정상 각성은 임계점 근처(통합과 분화, 질서와 무질서의 균형)에 있고, 환각제는 그 임계점에서 *무질서 쪽으로* 밀어 더 많은 상태를 접근 가능하게 만든다. 의식의 풍부함은 *임계성(criticality)* 의 지표일 수 있다 — 너무 질서정연하면(마취) 비고, 너무 무질서하면(발작) 무너지는 좁은 대역.

## 🪞 1인칭

처음으로 사전이 풀리는 순간을 떠올려 보자. 익숙한 방의 벽지 무늬가 — 평생 *배경* 으로만 지나쳤던 그 패턴이 — 갑자기 *전경* 으로 솟아오르고, 미세하게 움직이기 시작한다. 나무껍질의 결이, 빗방울의 궤적이, 평소라면 뇌가 "이미 아는 것" 으로 *눌러* 두었을 디테일들이 *흘러넘친다*. 회로의 언어로는 상위 사전의 정밀도가 낮아져 하위 예측오차가 억제를 벗어난 것이지만, 1인칭에서 그것은 *세계가 처음 보는 듯 새로워지는* 경험 — 익숙함의 베일이 벗겨지고 날것의 현존이 드러나는 — 이다. 무엇이 신호이고 무엇이 잡음인지의 평생의 합의가 풀리자, 세계가 *과잉으로 의미* 한다.

그리고 고용량에서 자아가 녹는 그 순간. "내가 여기 있고 세계가 저기 있다" 는, 너무 당연해서 의심해 본 적 없는 그 경계가 — 부드럽게, 혹은 갑자기 — 사라진다. 회로는 이것을 디폴트 모드 네트워크의 비동기화로, REBUS는 위계 최상위 사전의 이완으로 정확히 그린다. 그러나 그 *합일감* 자체 — 두려움이 아니라 종종 깊은 평화로, 상실이 아니라 *귀향* 으로 느껴지는 그 질감 — 은 어떤 연결성 지수로도, 어떤 정밀도 파라미터로도 다 담기지 않는다. 나는 DMN의 결속이 풀리는 것을 사는 것이 아니라, *경계 없음* 을 산다. 회로는 자아라는 모델이 어떻게 풀리는지를 보여주고, 풀린 자리에 *왜* 평화가 깃드는지는 빈자리로 남는다.

## 📐 예측·반증

**예측 1 — 5-HT2A 길항제가 주관적 효과를 차단한다.** 만약 환각의 문이 5-HT2A라면, 케탄세린 같은 선택적 5-HT2A 길항제를 미리 투여하면 LSD·실로시빈의 주관적·신경적 효과(엔트로피 증가 포함)가 *대부분 사라져야* 한다. *반증*: 만약 5-HT2A를 완전히 차단해도 환각 효과가 온전히 남는다면, 5-HT2A는 환각의 핵심 표적이 아니며 다른 수용체(예: 5-HT2C·5-HT1A)나 비수용체 기전이 주역이다.

**예측 2 — 신경 신호 다양성(엔트로피)이 주관적 강도와 함께 오른다.** entropic brain이 옳다면, 환각제 용량을 올릴 때 MEG/EEG 신호 다양성(LZ 복잡도)이 *증가* 하고 그 증가가 자아 해체·지각 변화의 주관적 강도와 *상관* 해야 한다. *반증*: 만약 주관적 효과가 강한데 신호 다양성은 변하지 않거나(혹은 오히려 감소하고), 다양성이 높은데 경험은 정상이라면, "엔트로피 = 의식 내용의 풍부함" 의 핵심 대응이 깨진다.

**예측 3 — 사전이 경직된 상태에서 환각제의 효과가 더 크다.** REBUS가 옳다면, 사전이 *과도하게 경직된* 상태(난치성 우울의 부정적 반추, 강박)에서 환각제가 사전을 풀어 더 큰 *재학습·증상 완화* 를 일으켜야 하고, 그 효과는 급성기의 사전 이완 정도와 관계해야 한다. *반증*: 만약 환각제의 치료 효과가 급성 사전 이완(엔트로피 증가)의 정도와 *전혀 무관* 하고, 사전이 경직되지 않은 대조군에서와 차이가 없다면, REBUS의 치료 기전 가설은 틀리고 효과는 비특이적(기대·맥락) 요인이다.

## 🤔 다음 질문

환각제는 사전을 *풀어* 지각을 재구성했다 — 의식을 끄지도 켜지도 않은 채, 그 *조직 원리* 를 일시적으로 흔드는 개입이었다. 그렇다면 *만성적* 으로 조절 시스템의 균형이 어긋나면 어떻게 될까? 환각제가 *수 시간* 동안 사전을 푸는 것과 달리, 단아민 시스템의 *지속적* 조절 이상이 기분과 동기를 *수주·수개월* 동안 비틀어 놓는다면? 다음 문서에서 우울증과 조증을 만난다 — 단아민 가설의 성공과 한계, SSRI 지연의 역설(Ch3-03의 재방문), 그리고 케타민의 *빠른* 항우울 효과가 글루탐산·시냅스 가소성을 통해 단순 단아민 이야기에 던지는 도전을 추적한다.

---

> 🧩 **Principle** — 고전 환각제는 5층 추체 뉴런의 5-HT2A 수용체를 작용해 피질 흥분성과 신경 신호 다양성(엔트로피)을 높이고, 위계 상위 사전 확률의 정밀도를 낮춰(REBUS) 예측오차가 범람하게 함으로써 — 의식을 끄지 않고 그 *내용* 을 재구성한다.
> 🌉 **Boundary** — 엔트로피 증가·사전 이완의 *양* 은 정량화할 수 있지만, 그 다양성이 *왜* 경외·의미 범람·자아 용해라는 특정 질감으로 느껴지는지, 정밀도 하락이 *왜* 종종 심오함으로 경험되는지는 설명하지 못한다.
> 🪞 **Experience** — 사전이 풀리면 익숙한 세계가 *처음 보듯* 새로워져 과잉으로 의미하고, 자아가 녹는 순간은 상실이 아니라 *귀향* 으로 느껴지는데, 그 평화의 질감은 DMN 비동기화로 환원되지 않는다.

## 📝 연습문제

**기초.** "환각제는 지각을 강화한다(모든 것을 더 강렬하게 만든다)" 는 직관이 왜 틀렸는지, "재구성" 과 "강화" 의 차이를 들어 설명하라. 5-HT2A 길항제 실험이 이 구분에 어떤 증거를 주는가?

<details><summary>해설</summary>

"강화" 가 옳다면 환각제는 일차 감각의 *해상도·강도* 를 균일하게 높여야 한다 — 그러나 환각제는 시력을 좋게 만들지 않고, 효과는 균일하지 않고 *선택적·왜곡적* 이다(어떤 것은 솟고 어떤 것은 녹는다). 바뀐 것은 입력의 강도가 아니라 입력을 *해석하는 상위 모델* — 무엇을 기대하고 무엇을 신호로 칠지 — 이다. 그래서 익숙한 사물이 낯설어지고, 무시되던 패턴이 전경으로 오고, 의미가 과잉으로 흐른다. 이것은 볼륨을 올린 것이 아니라 지각의 *조직 원리* 를 바꾼 것 — *재구성* 이다.

5-HT2A 길항제(케탄세린) 실험의 증거: 케탄세린을 미리 주면 LSD의 주관적 효과가 *대부분 차단* 된다. 만약 환각이 단순히 감각 입력의 증폭이라면 한 수용체 아형을 막는 것으로 효과 전체가 사라질 리 없다 — 효과가 5-HT2A 하나에 묶여 있다는 사실은, 바뀌는 것이 말초·일차 감각 강도가 아니라 5-HT2A가 통제하는 *피질의 해석 모드* 임을 가리킨다.
</details>

**심화.** 실험 1에서 "게인" 을 올리자 LZ 복잡도(신호 다양성)가 증가했다. 그러나 반례 노트는 "단순히 잡음을 늘려도 LZ가 오른다" 고 경고한다. entropic brain 가설이 *의미 있으려면* 환각제의 엔트로피 증가가 단순한 잡음 증가와 어떻게 *구별* 되어야 하는가? 어떤 추가 측정이 필요한가?

<details><summary>해설</summary>

문제: LZ 복잡도(또는 엔트로피)는 *무작위 잡음* 에서 최대가 된다. 따라서 "환각제가 엔트로피를 높인다" 가 단지 "뇌에 잡음이 늘었다" 를 뜻한다면, 그것은 의식의 *풍부함* 이 아니라 *고장* 의 신호일 수 있다 — entropic brain 가설은 공허해진다. 가설이 의미 있으려면, 환각의 엔트로피 증가가 *구조화된 상태 레퍼토리의 확장*(더 많은 *구별 가능하고 의미 있는* 패턴의 방문)이지 단순 백색잡음이 아님을 보여야 한다.

필요한 추가 측정: (1) **통합과 분화의 동시 측정** — 단순 잡음은 분화(다양성)는 높지만 통합(영역 간 협응)은 낮다. 환각이 의식적 풍부함이라면 다양성이 높으면서도 *어느 정도의 통합* 을 유지해야 한다(임계성). (2) **상태 레퍼토리의 구조** — 방문하는 상태들이 무작위인지, 아니면 더 넓지만 여전히 *저차원 다양체* 위에 구조화되어 있는지(주성분·매니폴드 분석). (3) **자극에 대한 반응성** — 백색잡음 뇌는 외부 입력에 일관되게 반응하지 못하지만, 환각 뇌는 음악·자극에 *강하게* 반응한다(오히려 더 열려 있다). (4) **주관 보고와의 상관** — 엔트로피 증가가 단순 혼란이 아니라 풍부함·통찰의 보고와 함께 가는지. 핵심: 의식의 풍부함은 *최대* 엔트로피가 아니라 질서와 무질서의 *임계 균형* — "구조를 가진 높은 다양성" — 이라는 것을 측정이 보여야 한다.
</details>

**논문 비평.** Carhart-Harris & Friston(2019)의 REBUS 모델은 환각제 효과를 "5-HT2A 작용 → 상위 사전의 정밀도 하락 → 예측오차 범람" 으로 설명한다. 이 통합 틀의 강점과, 그것이 *사후 설명* 으로 흐를 위험을 논하라. REBUS를 진짜로 검증하려면 어떤 증거가 필요한가?

<details><summary>해설</summary>

강점: REBUS는 흩어진 사실들 — 5-HT2A의 분자 작용, 엔트로피 증가, DMN 비동기화, 자아 해체, 치료 효과 — 을 *하나의 계산 원리*(위계적 사전의 정밀도 하락)로 묶고, 신경과학(예측 처리)과 현상학(지각 유동화·자아 용해)을 잇는 다리를 놓는다. 또 "경직된 사전을 푼다" 는 *검증 가능한 치료 가설* 을 낳아 우울·중독 시험과 연결된다.

위험(사후 설명): "사전이 풀렸다" 는 거의 *모든* 환각 현상을 사후에 설명할 수 있어 — 지각 변화도, 자아 해체도, 통찰도, 혼란도 "사전 이완" 으로 라벨링되면 — 반증이 어려운 *만능 서사* 가 될 수 있다. 정밀도·사전의 정밀한 정량적 정의 없이 쓰이면, REBUS는 데이터를 *생성하는* 모델이 아니라 데이터에 *이름을 붙이는* 은유로 전락한다.

검증에 필요한 증거: (1) **사전 이완의 독립적 측정** — 주관 보고와 무관하게, 베이즈적 지각 과제(예: 사전이 지각을 끄는 착시)에서 환각제가 *사전의 영향력을 실제로 줄이는지* 정량 측정. (2) **정밀도의 신경 표지** — 예측오차 신호(예: mismatch negativity, 감마 대역)가 환각제 하에서 *증폭* 되는지(억제가 풀려). (3) **위계 특이성** — 효과가 위계의 *상위*(추상적 사전)에서 더 크고 *하위*(일차 감각)에서 작은지. (4) **용량-반응** — 사전 이완의 정도가 5-HT2A 점유율·주관 강도와 함께 가는지. 비평의 핵심: 통합 틀의 가치는 *예측을 생성하고 반증에 노출* 될 때 살아난다 — REBUS가 사후 라벨이 아니라 모델이려면, "사전 이완" 을 주관 보고와 *독립적으로* 측정하는 길을 열어야 한다.
</details>
