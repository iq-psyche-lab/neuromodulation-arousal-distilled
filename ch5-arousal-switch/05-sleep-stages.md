# 🧠 수면 단계와 신경조절 — NREM과 REM의 두 화학 모드

*수면은 하나의 "꺼짐" 이 아니라, 단아민이 약하게 켜진 NREM과 콜린성이 지배하며 단아민이 *완전히 침묵* 하는 REM이라는 두 화학 모드를 ~90분 주기로 오가는 진동이며, 이 신경조절 패턴의 차이가 꿈의 현상학을 빚는다.*

## 🎯 핵심 질문

앞 문서들에서 우리는 깸과 잠을 가르는 플립플롭을 보았다. 그러나 "잠" 은 단일 상태가 아니다. 잠 안에는 적어도 두 개의 근본적으로 다른 모드가 있다 — NREM(비-REM)과 REM(빠른 안구 운동). 그리고 이 둘의 차이는 단순히 "깊이" 의 차이가 아니라 *신경조절 화학의 종류* 가 통째로 바뀌는 차이다.

가장 놀라운 사실부터 보자. REM 수면 동안 LC(노르에피네프린)와 DRN(세로토닌) — 깨어 있을 때 피질을 떠받치던 단아민 핵들 — 이 *거의 완전히 발화를 멈춘다*. 동시에 콜린성 핵(LDT/PPT)은 깸에 맞먹게 *활발히* 발화한다. 그래서 REM의 뇌는 EEG상 깸과 비슷하게 활성화되어 있지만(그래서 "역설수면"), 그 화학적 조성은 깸과 정반대다 — 콜린성은 켜지고 단아민성은 꺼진 상태. 꿈은 바로 이 *특이한 화학 모드* 안에서 일어난다.

핵심 질문은 이것이다. **왜 잠은 한 모드가 아니라 NREM과 REM의 *교대* 인가?** 무엇이 LC·DRN을 REM에서 *침묵* 시키고 콜린성 핵을 *해방* 하는가? 이 "REM-off/REM-on" 시소는 어떤 동역학이길래 ~90분 주기를 만드는가? 그리고 REM의 신경조절 조성(콜린성↑·단아민성↓)은 *왜* 꿈을 깸의 사고와 그토록 다르게 — 생생하고, 비논리적이고, 기억되지 않게 — 만드는가? 이 문서는 Hobson & McCarley(1975)의 상호작용 모델로 NREM-REM 진동을 그리고, 신경조절 상태가 꿈의 질감을 어떻게 빚는지를 본다.

## 🌍 어디서 마주치나

- **밤의 수면 구조(sleep architecture)**: 하룻밤은 "깸—NREM(N1→N2→N3 서파)—REM" 의 주기를 ~90분 간격으로 4~6회 반복한다. 밤 전반부엔 깊은 서파(N3)가 많고, 후반부로 갈수록 REM이 길어진다 — 새벽 꿈이 더 생생하고 잘 기억되는 이유.
- **생생한 꿈과 그 비논리성**: REM의 꿈은 강렬한 시각 이미지·기이한 장면 전환·감정 과잉이 특징인데, 정작 그 안에서는 *이상함을 알아채지 못한다*. 깨고 나서야 "말도 안 되는 꿈이었네" 한다 — REM의 단아민 침묵이 비판적 자각·기억을 억누른 결과.
- **항우울제와 REM 억제**: SSRI·삼환계 항우울제는 세로토닌·노르에피네프린을 높여 — REM을 *억제* 한다(REM 잠복기 증가, REM 시간 감소). 단아민이 REM-off라는 사실의 약리적 증거다.
- **콜린성 약물과 꿈**: 아세틸콜린에스터레이스 억제제(예: 일부 치매약)는 ACh를 높여 *생생한 꿈·악몽* 을 부작용으로 일으킨다 — REM-on 콜린계를 강화한 결과.
- **REM 행동장애(RBD)**: REM의 근육 무긴장이 깨지면 환자가 꿈을 *행동으로 옮긴다*(소리치고 발길질). 파킨슨병의 조기 징후로 주목받는다 — REM 회로의 고장이 임상으로 드러나는 창.

## 🔍 직관의 함정

**함정 1 — "잠은 깊어질수록 한 방향으로 점점 더 꺼진다."**
직관은 수면을 "얕은 잠 → 깊은 잠" 의 단일 축으로, 깊이의 다이얼로 본다. 반증: 이 그림은 REM을 설명하지 못한다. REM은 NREM보다 "더 깊은" 잠이 아니다 — EEG는 오히려 깸에 가깝게 *탈동기화* 되어 있고(역설수면), 뇌 대사는 깸 수준이며, 깨우기는 오히려 어렵다. 수면은 *한 축의 깊이* 가 아니라 *두 개의 질적으로 다른 모드* (NREM·REM)의 교대다. "깊이" 다이얼은 NREM 내부(N1~N3)에만 적용되고, REM은 그 축 *바깥* 의 별개 상태다.

**함정 2 — "REM의 뇌는 깸의 뇌와 같다(둘 다 활성화되어 있으니까)."**
REM과 깸이 둘 다 EEG 활성·꿈/사고를 보이니 "비슷한 상태" 로 묶기 쉽다. 반증: 신경조절 화학이 *정반대* 다. 깸은 콜린성+단아민성이 *함께* 켜진 상태인데, REM은 콜린성만 켜지고 단아민성(NE·5-HT)은 *완전히 침묵* 한다. 이 차이가 결정적이다 — 단아민은 신호-잡음비·비판적 평가·기억 부호화를 떠받치므로, 그 침묵이 REM 꿈의 비논리성·망각·감정 과잉을 만든다. 같은 "활성화" 라도 *어떤 조절 물질이* 활성화하느냐가 경험의 질을 가른다.

**함정 3 — "REM과 NREM의 전환은 깸-잠 플립플롭과 같은 종류다."**
앞서 본 깸-잠 스위치가 이분 안정·히스테리시스였으니, NREM-REM도 같으리라 생각하기 쉽다. 반증: NREM-REM은 *진동(oscillation)* 이다 — 두 상태에 안정적으로 *머무는* 것이 아니라 ~90분 주기로 *끊임없이 오간다*. 깸-잠이 "한 상태를 고수하는 토글 스위치(이분 안정)" 라면, NREM-REM은 "두 상태를 주기적으로 순환하는 시계(극한 순환)" 다. 같은 상호 억제 구조라도, 시간 지연·자기 흥분 항이 더해지면 안정점이 *진동* 으로 바뀐다 — 다른 종류의 동역학이다.

## ⚙️ 회로·분자 분석

**분석 1 — REM-off 단아민과 REM-on 콜린: 발화 패턴의 역전.**
단일세포 기록(Hobson, McCarley, Aston-Jones 등)은 명확한 패턴을 보인다. **LC(NE)·DRN(5-HT) 뉴런** 은 깸에서 가장 빠르게 발화하고, NREM에서 느려지며, REM에서 *거의 0* 으로 침묵한다 — 이것이 "REM-off" 세포다. 반대로 **콜린성 LDT/PPT 뉴런** 은 깸과 REM 둘 다에서 활발히 발화하고 NREM에서 약해진다 — "REM-on" 세포(일부는 REM-선택적). 그래서 세 상태의 조절 조성은: 깸 = 단아민↑·콜린↑, NREM = 단아민↓·콜린↓(둘 다 약함), REM = 단아민↓↓·콜린↑↑. REM은 *콜린 단독 지배* 의 독특한 모드다. $\square$

**분석 2 — Hobson-McCarley 상호작용 모델.**
Hobson & McCarley(1975)는 이 시소를 *생태학의 포식자-피식자* 처럼 형식화했다. REM-on 콜린성 집단($C$)과 REM-off 단아민성 집단($A$)이 상호작용하는데: (i) $C$는 자기 자신을 흥분시키고 $A$도 흥분시킨다, (ii) $A$는 $C$를 *억제* 한다, (iii) $A$는 스스로 감쇠한다. 그러면 $C$가 커지면 $A$를 키우고, 커진 $A$가 $C$를 누르고, $C$가 줄면 $A$도 감쇠하고, 그러면 $C$가 다시 풀려나는 — *순환* 이 생긴다. Lotka-Volterra형 방정식으로:

$$\dot C = C\,(a - b\,A), \qquad \dot A = A\,(-c + d\,C)$$

여기서 $C$=콜린성(REM-on), $A$=단아민성(REM-off). 이 계는 고정점 주위를 *주기적으로 도는* 극한 순환에 가까운 진동을 낳고, 파라미터를 맞추면 주기가 ~90분이 된다. NREM-REM 교대는 안정 상태가 아니라 *시계* 다. $\square$

**분석 3 — REM 무긴장: 꿈을 행동에서 분리하는 안전장치.**
REM에서 콜린성 활성은 피질을 깸처럼 켜지만, 동시에 연수(특히 SLD/subcoeruleus → 척수 억제성 뉴런) 경로가 척수 운동뉴런을 *글리신·GABA로 강하게 억제* 해 골격근을 마비시킨다 — REM 무긴장(atonia). 이것은 버그가 아니라 *기능* 이다: 생생한 꿈의 운동 명령이 실제 근육으로 나가지 못하게 막는 안전장치. 이 회로가 고장 나면 REM 행동장애(RBD)로 꿈을 행동화하고, 거꾸로 이 무긴장이 깸에 *침입* 하면 기면증의 탈력발작(→ Ch5-04)이 된다. 같은 무긴장 회로의 정상·과잉·결핍이 세 가지 다른 임상을 만든다. $\square$

**분석 4 — 왜 REM 화학이 꿈의 현상학을 빚는가.**
REM의 신경조절 조성(콜린↑·단아민↓)은 꿈의 특징적 질감과 인과적으로 연결된다. (i) **콜린성 우세** → 피질·해마 활성화, 내부에서 생성된 감각 이미지의 폭발(생생함). (ii) **노르에피네프린 침묵** → 신호-잡음비·"새것 탐지" 의 약화로 비논리적 장면 전환이 *이상하게 느껴지지 않음*. (iii) **세로토닌 침묵** → 비판적 평가·억제의 약화로 감정·충동이 여과 없이 분출(감정 과잉). (iv) **단아민 침묵** → 기억 부호화의 손상으로 꿈이 깨면 *빠르게 망각*(NE·ACh가 부호화를 여는 Ch4-04의 역). 즉 REM 꿈의 4대 특징 — 생생함·비논리·감정 과잉·망각 — 은 각각 특정 조절 물질의 켜짐/꺼짐에 대응한다. AIM 모델(Hobson)이 이를 활성화(A)·입력 게이팅(I)·조절(M) 3축으로 정리한다. $\square$

## 🧪 실험·임상 증거

### 실험 1 — 상호작용 모델로 NREM-REM ~90분 진동 재현

Hobson-McCarley 상호작용 모델을 적분해 콜린성(REM-on)과 단아민성(REM-off)이 *번갈아* 진동하며 ~90분 주기를 만드는 것을 본다. 핵심 예측: 두 집단은 *위상이 어긋난* 채 순환하고(한쪽이 정점일 때 다른 쪽은 상승 중), 주기는 파라미터로 조율된다. REM(콜린성 정점·단아민성 저점)이 주기적으로 재방문됨을 확인한다.

```python
import numpy as np
from scipy.integrate import odeint

def reciprocal_interaction(y, t, a, b, c, d):
    """Hobson-McCarley 상호작용 모델 (Lotka-Volterra형).
       C = 콜린성 REM-on (LDT/PPT),  A = 단아민성 REM-off (LC/DRN).
       콜린성은 스스로/단아민성을 흥분, 단아민성은 콜린성을 억제·자기감쇠."""
    C, A = y
    dC = C * (a - b * A)      # ACh 성장, 단아민성이 억제
    dA = A * (-c + d * C)     # 단아민성 감쇠, ACh가 구동
    return [dC, dA]

# Lotka-Volterra 소진동 주기 ≈ 2π/√(a·c). 90분이 되도록 a=c 설정.
a = c = 2 * np.pi / 90.0
b = d = 0.07
t = np.linspace(0, 400, 8000)              # 분 단위
sol = odeint(reciprocal_interaction, [1.2, 0.8], t, args=(a, b, c, d))
C, A = sol[:, 0], sol[:, 1]

# 콜린성(REM-on) 정점 = REM 에피소드. 주기 측정.
peaks = [i for i in range(1, len(C) - 1) if C[i] > C[i-1] and C[i] > C[i+1]]
period = np.mean(np.diff(t[peaks])) if len(peaks) >= 2 else float('nan')
print(f"콜린성(REM-on) 진동 주기 ≈ {period:.1f} 분   (REM 에피소드 {len(peaks)}회)")
print(f"콜린성 C 범위: {C.min():.2f} ~ {C.max():.2f}")
print(f"단아민성 A 범위: {A.min():.2f} ~ {A.max():.2f}")

# 위상 관계: C가 정점일 때 A의 값 (어긋난 위상 확인)
ip = peaks[1]
print(f"\nC 정점 시점 t={t[ip]:.0f}분: C={C[ip]:.2f}(높음=REM), A={A[ip]:.2f}")
# REM 비율: C가 평균 위일 때를 'REM 유사'로 표시
rem_like = np.mean(C > C.mean())
print(f"REM 유사 구간 비율 ≈ {rem_like:.2f}  (단아민성 침묵·콜린성 우세 시간)")
```

실행하면 콜린성(REM-on)과 단아민성(REM-off)이 *번갈아* 오르내리며 — 한쪽이 정점일 때 다른 쪽은 상승/하강 중인 *어긋난 위상* — ~90분 주기로 진동함을 본다. 콜린성이 정점인 구간이 REM 에피소드(단아민 침묵·콜린 우세)에 해당하고, 이 구간이 주기적으로 재방문된다. 깸-잠 플립플롭(이분 안정·정지)과 달리 NREM-REM은 *극한 순환에 가까운 시계* 임이 드러난다. 이는 Hobson & McCarley(1975, *Science*)의 상호작용 모델, REM-off 세포의 침묵을 보인 Aston-Jones & Bloom(1981)의 LC 기록, 콜린성 REM-on 세포 기록(McCarley 등)과 일치한다.

**반례 노트**: 순수 Lotka-Volterra 모델은 *중립 안정* 진폭을 가져 초기 조건에 따라 진폭이 달라지고 감쇠/발산하지 않는다 — 실제 NREM-REM은 항상성 압력·일주기 입력에 의해 진폭과 주기가 *조절* 되는 안정한 극한 순환에 더 가깝고(McCarley & Massaquoi의 후속 limit-cycle 모델이 이를 보강), 본 모델은 그 핵심 *상호작용 구조* 만 추상화했다. 또 단아민 핵을 *완전히 0* 으로 보는 것은 단순화로, REM에서 LC·DRN은 거의(완전히는 아님) 침묵하며 미세한 잔류 발화가 있다. 마지막으로 ~90분 주기는 종·개체차가 크고(쥐는 훨씬 짧다), 모델의 90분은 인간 평균에 맞춘 파라미터 선택이지 회로에서 *유도* 된 값이 아니다.

## 🌉 설명적 간극

✅ **회로가 설명하는 것**: 왜 잠이 한 모드가 아니라 NREM-REM 교대인가(상호작용 진동), 왜 REM에 단아민이 침묵하고 콜린이 지배하는가(REM-off/on 세포), 왜 항우울제가 REM을 억제하는가(단아민↑), 왜 꿈을 행동으로 옮기지 않는가(무긴장 안전장치), RBD·탈력발작이 같은 회로의 다른 고장임.

✅ **계산이 환원하는 것**: NREM-REM 주기를 콜린성-단아민성 상호작용의 극한 순환으로, 꿈의 4대 특징(생생·비논리·감정·망각)을 특정 조절 물질의 켜짐/꺼짐으로 대응시켜 정량화할 수 있다는 것.

❌ **남는 것**: REM의 화학 조성이 꿈의 *형식적* 특징(생생함·비논리)을 설명한다 해도, 꿈의 *구체적 내용* — *왜 하필 그 얼굴, 그 장소, 그 이야기* 가 펼쳐지는가 — 은 콜린성·단아민성 수치로 환원되지 않는다. 회로는 꿈의 *문법* 을 그리지만 꿈의 *시(詩)* 는 그리지 못한다.

❌ **또 남는 것**: 꿈은 분명 *경험* 이다 — 깊은 서파수면의 경험적 공백과 달리, 꿈에는 1인칭의 생생한 현존이 있다. 같은 "잠" 인데 왜 NREM에는 경험이 거의 없고 REM에는 *세계가 통째로* 있는가? 콜린성 우세가 그 차이와 상관한다 해도, 왜 그 화학 조성에서 *경험이 켜지는지* — 조성과 현상적 현존의 연결 — 은 회로 너머에 있다.

## 🧬 횡단 원리

**상호 억제 + 자기 흥분 = 진동** 은 신경계 전반의 리듬 발생 원리다. 호흡 리듬, 보행의 다리 교대, 씹기, 심장의 페이스메이커 — 모두 *서로 억제하는 두 집단에 시간 지연이나 자기 구동이 더해지면* 정지가 아니라 *순환* 이 나온다. 깸-잠 플립플롭(이분 안정)과 NREM-REM 진동(극한 순환)은 *같은 상호 억제 모티프* 의 두 동역학 체제다 — 파라미터에 따라 같은 회로가 "토글" 도 "시계" 도 될 수 있다는 것은, 신경계가 소수의 구조 원리로 다양한 시간 패턴을 빚는 방식을 보여 준다.

여기서 더 깊은 횡단 원리: **조절 물질의 *조합* 이 의식의 *모드* 를 정의한다**. 깸·NREM·REM은 단순히 "각성 수준" 의 높낮이가 아니라, 콜린성·단아민성이라는 두 축의 *서로 다른 조합* 이다 — (켜짐,켜짐)=깸, (꺼짐,꺼짐)=NREM, (켜짐,꺼짐)=REM. 의식의 상태 공간은 1차원 다이얼이 아니라 *여러 조절 축이 펼치는 다차원 공간* 이고, 우리가 경험하는 각 "상태" 는 그 공간의 한 영역이다(→ Ch6 환각제는 또 다른 축인 5-HT2A를 켜서 *제4의 영역* 으로 간다). 신경조절은 의식의 *좌표계* 를 제공하고, 각 좌표에 다른 질감의 경험이 깃든다 — 이 레포 전체가 추적하는 "상태 → 질감" 의 지도가 여기서 다차원으로 펼쳐진다.

## 🪞 1인칭

REM 꿈 한복판에 있을 때, 가장 기이한 것은 *이상함을 알아채지 못한다는* 것이다. 죽은 사람과 태연히 대화하고, 방이 바다로 변하고, 내가 날고 있어도 — 꿈 안의 나는 그것을 의심하지 않는다. 깨고 나서야 "말도 안 됐네" 하며 웃는다. 이 *비판적 자각의 부재* 를, 회로는 "단아민(NE·5-HT)의 침묵으로 신호-잡음 평가와 억제가 꺼졌다" 고 정확히 그린다. 나는 꿈속에서 *덜 회의적인 뇌* 를 가지고 있었던 것이다. 그리고 그 비논리의 강물이 깨어나는 순간 손가락 사이로 빠져나가 거의 남지 않는 것 — 그 망각 — 도 같은 침묵(부호화의 닫힘)의 결과다.

그러나 정직해야 할 자리가 여기에 있다. 회로는 꿈이 *왜 생생한지, 왜 비논리적인지, 왜 잊히는지* 를 설명한다. 그러나 어젯밤 내가 꾼 꿈의 *그 특정한 풍경* — 오래전 떠난 친구의 얼굴, 한 번도 가본 적 없는데 익숙했던 그 골목, 깨어나서도 가슴에 남은 그 알 수 없는 그리움 — 은 콜린성 수치나 단아민 침묵으로는 한 글자도 설명되지 않는다. 신경조절은 꿈이라는 무대의 *조명과 물리법칙* 을 정한다. 무대 위에 *어떤 이야기가 상연되는가* 는 다른 층위의 물음이고, 더 깊게는 — 깊은 잠의 공백과 달리 — 꿈에는 *누군가 그것을 겪고 있다는* 현존이 있다는 사실, 그 경험의 *있음* 자체가 가장 환원되지 않는 잔여다. 이 문서는 잠의 화학 모드를 그렸고, 그 모드 안에서 *펼쳐지는 세계와 그것을 겪는 나* 는 빈자리로 남긴다.

## 📐 예측·반증

**예측 1 — 단아민 증가는 REM을 억제한다.** REM-off가 단아민성이라면, NE·5-HT를 약리적으로 높이는 약물(SSRI·SNRI·MAO 억제제)은 REM 잠복기를 늘리고 REM 시간을 *줄여야* 한다. *반증*: 만약 단아민을 높이는 약물이 REM에 영향이 없거나 *오히려 늘린다면*, "단아민=REM-off" 모델은 틀린다. (실제로 SSRI의 강한 REM 억제는 이 예측을 지지한다.)

**예측 2 — NREM-REM은 정지가 아니라 진동이다.** NREM-REM 교대는 안정 상태에 머무는 토글이 아니라 ~주기적 순환이므로, 외란 후에도 시스템은 *같은 주기로 복귀* 하고 두 신경조절 집단은 *어긋난 위상* 으로 진동해야 한다. *반증*: 만약 수면이 외부 자극 없이는 한 모드(NREM 또는 REM)에 *무한정 머문다면*, 그것은 진동이 아니라 이분 안정이고 상호작용-시계 모델은 틀린다.

**예측 3 — REM 무긴장 회로의 고장은 양방향 임상을 만든다.** 무긴장이 *결핍* 되면(깸엔 정상, REM에 마비 실패) 꿈을 행동화하는 RBD가, 무긴장이 *깸에 침입* 하면 탈력발작이 나와야 하며 — 둘 다 같은 연수-척수 억제 회로를 가리켜야 한다. *반증*: 만약 RBD와 탈력발작의 무긴장이 *서로 다른* 독립 회로로 밝혀진다면, "하나의 무긴장 회로, 세 가지 상태(정상·결핍·침입)" 라는 통합 모델은 틀린다.

## 🤔 다음 질문

우리는 이제 의식의 상태 공간이 1차원 다이얼이 아니라 *여러 조절 축이 펼치는 다차원 공간* 임을 보았다 — 깸·NREM·REM은 콜린성·단아민성 두 축의 서로 다른 조합이었다. 그런데 이 공간에는 더 많은 영역이 있다. 마취약이 *시상-피질 통합* 을 끊어 의식을 끌 때, 신경조절 좌표계에서 무엇이 꺼지는가? 환각제가 *5-HT2A* 라는 또 다른 축을 켜서 깸도 꿈도 아닌 *제4의 영역* 으로 갈 때, 지각은 어떻게 재구성되는가? 그리고 — 각성·수면·REM·마취·도취가 모두 신경조절 좌표로 그려진다 해도 — 왜 그 각 좌표에 *특정 질감의 경험* 이 깃드는가? 다음 챕터에서 마취와 신경조절을 만나며, 각성 스위치에서 의식 자체의 ON/OFF로 건너간다.

---

> 🧩 **Principle** — 수면은 단일 깊이가 아니라 단아민이 약한 NREM과 콜린성이 지배하며 단아민이 침묵하는 REM의 ~90분 교대이며, REM-on 콜린성과 REM-off 단아민성의 상호작용이 극한 순환(시계)을 이뤄 이 조성 차이가 꿈의 생생함·비논리·감정·망각을 빚는다.
> 🌉 **Boundary** — REM 화학이 꿈의 *형식*(생생·비논리·망각)을 설명해도, 꿈의 *구체적 내용* 과 — NREM의 공백과 달리 — 꿈에 *경험이 있다는* 현존 자체는 조절 물질 수치로 환원되지 않는다.
> 🪞 **Experience** — 꿈 안에서는 비논리를 *이상하게 느끼지 못하고*(단아민 침묵), 깨면 강물처럼 빠져나가 망각되며, 그럼에도 깊은 잠의 공백과 달리 *누군가 그것을 겪고 있다는* 현존이 남는다.

## 📝 연습문제

**기초.** REM 수면이 "역설수면" 이라 불리는 이유를 EEG와 신경조절 화학의 관점에서 설명하라. 또 깸·NREM·REM 세 상태를 (콜린성, 단아민성)의 켜짐/꺼짐 조합으로 정리하라.

<details><summary>해설</summary>

"역설" 인 이유: REM의 EEG는 깸과 비슷하게 *탈동기화(빠른 저진폭)* 되어 있고 뇌 대사도 깸 수준으로 높은데 — 즉 뇌는 "깬 것처럼 활성화" 되어 있는데 — 정작 몸은 마비(무긴장)되어 있고 행동상으로는 가장 *깊이* 잠들어(깨우기 어려움) 있다. "활성화된 뇌 + 마비된 몸 + 깊은 잠" 이라는 역설적 조합이라 역설수면이다.

세 상태의 조합:
- **깸**: 콜린성 ON, 단아민성 ON (둘 다 켜짐)
- **NREM**: 콜린성 OFF(약함), 단아민성 OFF(약함) — 둘 다 가라앉음
- **REM**: 콜린성 ON(강함), 단아민성 OFF(침묵) — 콜린 단독 지배

핵심은 REM이 깸과 *다른* 화학(단아민 침묵)을 가진다는 점이며, 이것이 꿈의 비논리·망각·감정 과잉의 원천이다.
</details>

**심화.** 실험 1의 상호작용 모델 $\dot C = C(a-bA)$, $\dot A = A(-c+dC)$에서, 왜 이 계가 *정지*(고정점 정착)가 아니라 *진동* 을 내는지를 포식자-피식자 비유로 설명하라. 깸-잠 플립플롭(이분 안정)과 이 NREM-REM 진동이 같은 "상호 억제" 인데도 다른 동역학을 내는 이유는?

<details><summary>해설</summary>

진동의 원리(포식자-피식자 비유): $C$(콜린성)를 *피식자*, $A$(단아민성)를 *포식자* 로 보자. (1) $A$가 적으면 $C$는 억제를 덜 받아 증가한다(피식자 번성). (2) $C$가 많아지면 $A$의 구동($+dC$)이 커져 $A$가 증가한다(포식자 번성). (3) $A$가 많아지면 $C$를 강하게 억제해 $C$가 감소한다(피식자 감소). (4) $C$가 줄면 $A$의 먹이가 줄어 $A$도 감쇠한다($-c$). → 그러면 다시 (1)로. 이 *지연된 되먹임* 의 고리가 두 변수를 어긋난 위상으로 *돌게* 만들어 정지하지 못하고 순환한다.

같은 상호 억제인데 다른 이유: 깸-잠 플립플롭은 두 변수가 *서로* 억제하고 *각자 감쇠* 하는 대칭 구조 → 한쪽이 이기면 굳어지는 *이분 안정*. NREM-REM은 한쪽($C$)이 상대($A$)를 *흥분* 시키고($+dC$), 상대($A$)가 그것을 억제하는 *비대칭 + 자기 구동* 구조 → 닫히지 않는 되먹임 고리 → *진동*. 즉 같은 "상호작용" 이라도 부호 구조(상호 억제 vs 흥분-억제 혼합)와 자기 항이 동역학 체제(고정점 vs 극한 순환)를 가른다.
</details>

**논문 비평.** Hobson & McCarley(1975)의 상호작용 모델은 REM을 "콜린성 REM-on과 단아민성 REM-off의 시소" 로 우아하게 설명했고, 이를 근거로 꿈을 "뇌간이 무작위로 피질을 자극하면 피질이 사후에 이야기를 지어내는 것" 으로 본 활성화-합성(activation-synthesis) 가설을 내놓았다. 이 가설의 강점과, 꿈의 *내용* 에 대해 그것이 과소평가한 것은 무엇인가?

<details><summary>해설</summary>

강점: (1) 꿈을 *프로이트적 상징 해독* 같은 검증 불가능한 틀에서 *신경생리* 의 영역으로 끌어왔다 — 꿈의 형식적 특징(생생함·비논리·기이한 전환·감정 과잉)을 REM의 신경조절 조성(콜린↑·단아민↓)으로 *기계론적* 으로 설명. (2) 검증 가능한 예측(REM-off 세포의 침묵, 항우울제의 REM 억제)을 낳아 데이터로 지지받았다. (3) 무긴장·안구운동 같은 동반 현상까지 한 회로로 묶었다.

과소평가한 것: 활성화-합성은 꿈을 "무작위 뇌간 신호의 사후 합성" 으로 보아 꿈의 *내용* 을 거의 우연·잡음으로 취급한다. 그러나 후속 연구는 꿈 내용이 완전히 무작위가 아님을 시사한다 — 최근 경험·정서적 관심사·기억 공고화와 *체계적* 으로 연관되고(낮의 사건이 꿈에 반영), 전뇌(특히 변연계·시각연합피질)의 능동적 기여가 크다(Solms는 도파민성 동기 회로의 역할을 강조하며 순수 뇌간 기원설을 비판). 비평의 핵심: 상호작용 모델은 꿈의 *형식*(왜 생생하고 비논리적인가)을 훌륭히 설명하지만, 꿈의 *내용*(왜 하필 그 내용인가)과 그 *의미·정서적 일관성* 은 "무작위 + 합성" 으로 환원되지 않으며 — 이는 이 문서의 설명적 간극(꿈의 문법은 설명하나 꿈의 시는 설명 못 함)과 정확히 같은 한계다.
</details>
