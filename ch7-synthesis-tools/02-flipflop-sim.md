# 🧠 플립플롭 각성 모델 — 두 안정 상태를 코드로 적분하기

*Chapter 5 에서 nullcline 으로 그린 상호 억제 스위치를 이제 scipy 로 직접 적분해, 강한 결합에서 깸·잠 두 끌개가 공존하고 결합을 낮추면(오렉신 손실) 스위치가 깜빡이며 무너지는 — 기면증의 회로적 핵심을 우리 손으로 재현한다.*

## 🎯 핵심 질문

Chapter 5 의 03 문서는 각성 핵($A$)과 VLPO($S$)의 상호 억제가 *숨은 양성 되먹임* 을 만들어 중간 상태를 불안정(안장)하게 하고 깸·잠 두 끌개를 안정화함을 nullcline 으로 *그렸다*. 그리고 04 문서는 오렉신 손실이 그 스위치를 불안정하게 만들어 기면증을 일으킨다고 *주장* 했다. 이제 우리는 그 그림과 주장을 *실행되는 미분방정식* 으로 옮긴다.

핵심 질문은 두 갈래다. 첫째, **상호 억제 2변수 ODE 를 직접 적분하면, 정말로 *같은 입력* 인데도 초기 조건에 따라 깸 또는 잠으로 갈리는 이분 안정이 나타나는가 — 그리고 수면 압력(아데노신)을 천천히 올릴 때 어떤 문턱에서 *급격히* 잠으로 넘어가는 날카로운 전환이 보이는가?** 둘째, **상호 결합 강도 $w$ 를 낮추면(오렉신 뉴런 소실의 모델) 스위치가 어떻게 무너지는가 — 두 끌개가 하나로 합쳐지는가, 아니면 잡음 속에서 깸·잠을 *제멋대로 오가는 깜빡임*(narcoleptic flickering)이 나타나는가?**

이것은 계산 문서다. TD 의 이산 갱신과 달리 여기서는 *연속 시간* 의 동역학이 무대다. 우리는 두 실험에서, 첫째는 결정론적 ODE 로 이분 안정과 날카로운 전환을 보이고, 둘째는 잡음을 명시적으로 넣어 결합 약화가 안정한 스위치를 *불안정한 깜빡임* 으로 바꾸는 것을 본다.

## 🌍 어디서 마주치나

플립플롭 동역학은 추상적 수학이 아니라 우리의 수면 경험과 임상 곳곳에 새겨져 있다.

- **잠들 때의 "갑자기".** 책을 읽다 졸릴 때, 각성에서 수면으로의 전이는 점진적이라기보다 어느 순간 *툭* 넘어간다 — 우리 코드에서 두 끌개 사이를 빠르게 통과하는 궤적으로 나타난다.
- **수면 단편화와 노화.** 플립플롭이 약해지면(노화, 오렉신 손실) 밤새 깸과 잠을 여러 번 오간다 — 실험 2 의 결합 약화가 만드는 깜빡임이 임상에서 조각난 수면으로 나타난다.
- **기면증의 수면 발작.** 오렉신/히포크레틴 뉴런의 자가면역성 소실로 스위치의 걸쇠가 풀려, 깨어 있어야 할 시간에 수면(과 REM)이 *침입* 한다. 우리는 이를 $w$ 를 낮춘 잡음 ODE 로 재현한다.
- **마취 유도.** 마취약 용량을 천천히 올리면 의식은 선형으로 흐려지지 않고 어떤 문턱에서 *급격히* 꺼진다(→ Ch6) — 플립플롭형 날카로운 전환의 또 다른 얼굴이다.
- **양안 경쟁·의사결정.** 서로 억제하는 두 신경 집단이 중간을 금지하고 깨끗한 선택을 만드는 같은 회로 모티프가 지각·결정에서도 반복된다.

## 🔍 직관의 함정

**함정 1 — "각성-수면은 하나의 연속 변수(졸림 정도)를 천천히 적분하면 된다."**
직관은 각성을 0(깊은 잠)부터 1(완전 각성)까지의 아날로그 다이얼로 보고, 그 값을 시간에 따라 부드럽게 변화시키면 된다고 본다. 반증: 단일 변수 모델은 입력이 중간값일 때 *중간 출력* 을 안정적으로 낸다 — 그러나 실제 수면-각성은 중간에 머물지 않고 빠르게 통과한다. 코드에서 우리는 *두* 변수의 상호 억제가 같은 연속 입력을 *이산 두 상태* 로 양자화함을 본다. 적분기는 같아도(Euler/RK), 방정식의 *구조* 가 다르면 출력의 위상학이 근본적으로 다르다.

**함정 2 — "ODE 를 적분하면 항상 하나의 정상상태로 수렴한다."**
미분방정식이 안정점으로 수렴한다는 일반적 경험이, 그 안정점이 *유일* 하다는 오해를 낳는다. 반증: 비선형 ODE 는 여러 안정점(다중 안정성)을 가질 수 있고, *어느* 안정점에 도달하는지는 초기 조건이 정한다. 우리 코드에서 같은 파라미터·다른 초기 조건이 다른 끌개로 수렴하는 것을 직접 출력해 — "수렴 = 유일" 이라는 직관을 깬다. 끌개의 *개수* 는 적분이 아니라 nullcline 의 기하학이 정한다.

**함정 3 — "잡음을 넣으면 모델이 그냥 지저분해질 뿐, 새로운 현상은 없다."**
결정론적 모델에 노이즈는 군더더기처럼 보인다. 반증: 잡음은 *상태 전환의 타이밍을 결정하는 본질적 요소* 다. 깊은 끌개(강한 결합)에서는 잡음이 장벽을 거의 못 넘어 상태가 안정적이지만, 얕은 끌개(약한 결합)에서는 같은 잡음이 *자주* 장벽을 넘어 깜빡임을 만든다. 즉 기면증의 핵심 증상(불안정한 전환)은 *잡음 × 얕은 장벽* 의 상호작용이며, 잡음 없는 모델로는 이 확률적 전환의 빈도를 잴 수 없다. 잡음은 노이즈가 아니라 *신호* 다.

## ⚙️ 회로·분자 분석

**분석 1 — 상호 억제 2변수 ODE 의 코드 표상.** 각성 핵의 통합 활동을 $A$(LC·DRN·TMN 의 합), VLPO 활동을 $S$ 로 놓는다. 각자 자기 구동 입력($I_A, I_S$)을 받고, 상대로부터 *포화하는* 억제를 받으며, 시간상수 $\tau$ 로 감쇠한다:
$$\tau\dot A = I_A - F(S) - A, \qquad \tau\dot S = I_S - F(A) - S, \qquad F(x) = w\frac{x^n}{\theta^n + x^n}.$$
코드에서 $F$ 는 가파른 Hill 함수(큰 $n$)로 구현되어, 상대가 문턱 $\theta$ 를 넘으면 거의 *완전히* 억제한다. 이 가파름이 이분 안정의 원천이다. $\dot A, \dot S$ 를 우변 함수로 정의하고 Euler 또는 `scipy.integrate.odeint` 로 적분하면 궤적이 끌개로 굴러간다. $\square$

**분석 2 — 이분 안정의 코드적 검증: 초기 조건 분기.** 이분 안정의 *조작적 정의* 는 "같은 파라미터, 다른 초기 조건 → 다른 정상상태" 다. 코드에서 이를 직접 테스트한다: $(A_0, S_0) = (1.2, 0)$(깸에서 출발)과 $(0, 1.2)$(잠에서 출발)를 같은 $w, I_A, I_S$ 로 적분해, 전자가 $A$-높은 끌개로·후자가 $S$-높은 끌개로 수렴하면 이분 안정이 확인된다. 직관: 중간점에서 $A$ 가 우연히 조금 커지면 $F(A)$↑로 $S$↓ → $S$ 의 $A$ 억제↓ → $A$ 가 더 커진다(양성 되먹임). 작은 우위가 증폭되어 가까운 극단으로 굴러간다. $\square$

**분석 3 — 수면 압력 스윕과 날카로운 전환.** 수면 압력(아데노신 축적)을 $S$ 의 구동 $I_S$ 로 모델링한다. $I_S$ 를 천천히 올리며 각 값에서 정상상태 $A$ 를 기록하면, $A$ 는 선형으로 떨어지지 않고 *어떤 임계 $I_S$ 에서 급격히* 깸→잠으로 점프한다. 더 나아가 $I_S$ 를 올렸다 *내릴* 때 전환점이 달라진다 — 히스테리시스. 두 임계점 사이가 이분 안정 영역이고, 그 폭이 스위치의 "끈적함"(잡음 강건성)이다. 코드는 상승·하강 스윕을 각각 돌려 이 두 임계점과 히스테리시스 폭을 수치로 출력한다. $\square$

**분석 4 — 오렉신 손실 = 결합 약화 + 잡음.** 오렉신/히포크레틴은 외측 시상하부에서 각성 핵($A$)으로 흥분성 투사를 보내, 각성 끌개의 *장벽 높이* 를 키워 잡음이 스위치를 잘못 뒤집지 못하게 한다. 모델에서 이는 (i) $A$ 의 실효 구동을 약간 올리고, 더 본질적으로 (ii) 상호 결합 $w$/장벽을 높이는 *안정자* 효과다. 오렉신을 잃으면 실효 $w$ 가 낮아져 장벽이 얕아지고, 같은 크기의 잡음에도 스위치가 *자주* 뒤집힌다. 코드에서 우리는 결정론적 ODE 에 가우시안 잡음 항을 더한 *확률 미분방정식(SDE)* 을 Euler-Maruyama 로 적분해, 강한 결합(안정)과 약한 결합(깜빡임)의 전환 빈도를 직접 센다. $\square$

## 🧪 실험·임상 증거

이론의 원전은 **Saper, Chou & Scammell (2001), "The sleep switch: hypothalamic control of sleep and wakefulness", *Trends Neurosci.* 24:726** 와 **Saper et al. (2005, *Nature*)** 의 오렉신 안정자 역할이다. 아래 두 실험은 헤드리스로 실행되며(플롯은 `Agg`), 출력은 텍스트로도 확인된다.

### 실험 1 — 상호 억제 ODE: 이분 안정과 수면 압력에 따른 날카로운 전환

```python
import numpy as np
import matplotlib
matplotlib.use("Agg")            # 헤드리스
import matplotlib.pyplot as plt

def F(x, w, theta=0.4, n=4):
    """상대 핵 활동 x 의 포화 억제 (가파른 Hill 함수)."""
    x = max(x, 0.0)
    return w * x**n / (theta**n + x**n)

def settle(IA, IS, w, IC, T=80.0, dt=0.01, tau=1.0):
    """상호 억제 2변수 모델을 정상상태까지 Euler 적분.
       A=각성 핵(LC/DRN/TMN 합), S=VLPO."""
    A, S = IC
    for _ in range(int(T / dt)):
        dA = (IA - F(S, w) - A) / tau     # 각성: 구동 − VLPO 억제 − 감쇠
        dS = (IS - F(A, w) - S) / tau     # 수면: 구동 − 각성 억제 − 감쇠
        A = max(0.0, A + dt * dA)
        S = max(0.0, S + dt * dS)
    return A, S

# (1) 이분 안정: 같은 파라미터, 다른 초기 조건 → 다른 끌개
w_strong, IA, IS = 1.5, 0.8, 0.8
wake = settle(IA, IS, w_strong, (1.2, 0.0))   # 깸에서 출발
sleep = settle(IA, IS, w_strong, (0.0, 1.2))  # 잠에서 출발
print("[강한 결합 w=1.5 — 이분 안정 검증]")
print(f"  깸에서 출발 → (A={wake[0]:.3f}, S={wake[1]:.3f})  ← 각성 끌개")
print(f"  잠에서 출발 → (A={sleep[0]:.3f}, S={sleep[1]:.3f})  ← 수면 끌개")
print(f"  같은 입력인데 다른 정상상태 → 이분 안정 확인")

# (2) 수면 압력 스윕: 상승·하강 → 날카로운 전환 + 히스테리시스
IS_vals = np.linspace(0.2, 1.6, 29)
A, S = 1.2, 0.0                       # 깸에서 시작
up = []
for v in IS_vals:                     # 수면 압력 상승 스윕
    A, S = settle(IA, v, w_strong, (A, S), T=40); up.append(A)
down = []
for v in IS_vals[::-1]:               # 수면 압력 하강 스윕
    A, S = settle(IA, v, w_strong, (A, S), T=40); down.append(A)
down = down[::-1]
up, down = np.array(up), np.array(down)

sw_up = IS_vals[np.argmax(up < 0.3)]          # 깸→잠 점프하는 IS
sw_dn = IS_vals[::-1][np.argmax(down[::-1] > 0.3)]  # 다시 깸 되는 IS
print("\n[수면 압력(IS) 스윕 — 날카로운 전환 + 히스테리시스]")
print(f"  상승 스윕: IS≈{sw_up:.2f} 에서 깸→잠 급전환")
print(f"  하강 스윕: IS≈{sw_dn:.2f} 까지 잠 유지 (히스테리시스 폭 ≈ {abs(sw_up - sw_dn):.2f})")

# 전환의 날카로움: 인접 IS 간 A 변화의 최대 기울기
max_jump = np.max(np.abs(np.diff(up)))
print(f"  상승 스윕 최대 |ΔA| = {max_jump:.3f} (선형이면 작고, 날카로우면 큼)")

# (3) PNG: 히스테리시스 루프
fig, ax = plt.subplots(figsize=(6.5, 4.5))
ax.plot(IS_vals, up, "o-", label="상승 (깸→잠)", color="#c0392b")
ax.plot(IS_vals, down, "s-", label="하강 (잠→깸)", color="#2980b9")
ax.set_xlabel("수면 압력 I_S (아데노신)")
ax.set_ylabel("각성 핵 활동 A")
ax.set_title("플립플롭 히스테리시스 루프 (w=1.5)")
ax.legend(); fig.tight_layout()
fig.savefig("flipflop_hysteresis.png", dpi=120)
print("\n[저장] flipflop_hysteresis.png — 상승·하강 곡선이 갈라지는 히스테리시스 루프")
```

실행하면 (1) 강한 결합에서 *같은 입력* 인데도 출발점이 깸이면 $A\approx 0.8, S\approx 0$, 출발점이 잠이면 $A\approx 0, S\approx 0.8$ 로 — 두 뚜렷한 끌개가 공존함을 본다. (2) 수면 압력 $I_S$ 를 올릴 때 $A$ 는 선형으로 떨어지지 않고 어떤 임계값에서 *급격히* 점프하며(최대 $|\Delta A|$ 가 큼), 올림·내림의 전환점이 달라 히스테리시스 루프가 나타난다(`flipflop_hysteresis.png` 에 두 곡선이 갈라져 저장됨). 이는 Saper, Chou & Scammell(2001)의 sleep switch 가설을 동역학으로 재현한 것이다.

### 실험 2 — 결합 약화(오렉신 손실): 안정한 스위치 → 불안정한 깜빡임

```python
import numpy as np

def F(x, w, theta=0.4, n=4):
    x = np.maximum(x, 0.0)
    return w * x**n / (theta**n + x**n)

def simulate_sde(w, T=400.0, dt=0.01, tau=1.0, IA=0.7, IS=0.7,
                 noise=0.14, seed=0):
    """상호 억제 모델 + 가우시안 잡음 (Euler-Maruyama).
       w 가 크면 깊은 장벽(안정), 작으면 얕은 장벽(깜빡임)."""
    rng = np.random.default_rng(seed)
    n_steps = int(T / dt)
    A = np.zeros(n_steps); S = np.zeros(n_steps)
    A[0], S[0] = 1.0, 0.0                       # 깸에서 시작
    sq = np.sqrt(dt)
    for i in range(1, n_steps):
        dA = (IA - F(S[i-1], w) - A[i-1]) / tau
        dS = (IS - F(A[i-1], w) - S[i-1]) / tau
        A[i] = max(0.0, A[i-1] + dt*dA + noise*sq*rng.standard_normal())
        S[i] = max(0.0, S[i-1] + dt*dS + noise*sq*rng.standard_normal())
    return A, S

def count_transitions(A, thresh_hi=0.45, thresh_lo=0.25):
    """깸(A>hi)↔잠(A<lo) 상태 전환 횟수 (슈미트 트리거식 히스테리시스 카운트)."""
    state = "wake" if A[0] > thresh_hi else "sleep"
    n_tr = 0
    for a in A:
        if state == "wake" and a < thresh_lo:
            state = "sleep"; n_tr += 1
        elif state == "sleep" and a > thresh_hi:
            state = "wake";  n_tr += 1
    return n_tr

print("결합 강도 w 에 따른 깸↔잠 전환 횟수 (잡음 동일, 400 시간 단위)")
print(f"{'w':>6} {'전환 수':>8}  해석")
for w in [1.6, 1.2, 0.9, 0.7, 0.5]:
    A, S = simulate_sde(w, seed=1)
    n_tr = count_transitions(A)
    frac_wake = np.mean(A > 0.35)
    tag = "안정 (깸 고정)" if n_tr <= 2 else "깜빡임(narcoleptic flickering)"
    print(f"{w:>6.1f} {n_tr:>8}  깸 비율 {frac_wake:.2f}  {tag}")

print("\n→ 강한 결합(w≥1.2)에서는 잡음에도 깸 상태가 안정적으로 유지된다(전환 ≈ 0).")
print("→ 결합이 약해지면(w≤0.7, 오렉신 손실) 같은 잡음에도 깸↔잠을 빈번히 오간다.")
print("→ 이것이 기면증의 회로적 핵심: 장벽이 얕아져 잡음이 스위치를 자주 뒤집는다.")
```

실행하면 결합 $w$ 가 강할 때(1.2 이상) 깸 끌개의 장벽이 깊어 잡음이 거의 장벽을 못 넘어 전환 횟수가 0~2 에 머문다(안정). $w$ 를 0.7 이하로 낮추면(오렉신 손실 모델) 장벽이 얕아져 같은 크기의 잡음에도 깸↔잠을 빈번히 오가는 *깜빡임* 이 나타나 전환 횟수가 급증한다. 이것이 기면증에서 수면(과 REM)이 깨어 있는 시간으로 *침입* 하는 현상의 동역학적 핵심이며, Saper 등(2005)의 오렉신 안정자 역할과 일치한다 — 오렉신은 각성을 *만드는* 페달이 아니라 스위치를 제자리에 고정하는 *걸쇠* 다.

**반례 노트.** 이 모델은 각성 핵 전체를 단일 변수 $A$ 로 뭉쳤다. 실제로는 LC·DRN·TMN·콜린성 핵이 서로 다른 동역학과 REM 위상 의존성을 가지며(콜린성 핵은 REM 에서 오히려 켜짐 — Ch5-05), 단일 $A$ 로는 NREM-REM 내부 구조나 탈력발작(cataplexy, 깨어 있는 중 REM 근긴장 소실)의 메커니즘을 담지 못한다. 또 $F$ 의 가파름($n$)과 결합 $w$ 의 절대값은 직접 측정된 것이 아니라 이분 안정을 *낳도록* 선택된 것이다 — 진짜 회로가 이 형태의 비선형성을 갖는지는 부분적으로만 검증되었다. 잡음의 크기·색깔(white vs colored)도 자유 파라미터이며, 전환 빈도의 정량값은 이 선택에 의존한다. 모델은 *왜 기면증이 불안정한가* 의 구조를 보이지, 그 빈도의 절대값을 예측하지 않는다.

## 🌉 설명적 간극

✅ **코드가 설명하는 것**: 왜 "반쯤 깬" 안정 상태가 없는가(중간점이 안장), 왜 전환이 빠르고 깨끗한가(끌개 사이 점프), 왜 한 번 정한 상태가 잡음에 강건한가(히스테리시스), 오렉신이 왜 안정자인가(장벽 높이), 기면증이 왜 *불안정* 인가(약한 결합 × 잡음).

✅ **코드가 환원하는 것**: 각성-수면 전환을 상호 억제 2변수 ODE 의 이분 안정·분기로, 스위치의 끈적함을 히스테리시스 폭으로, 오렉신 효과를 결합 강도 $w$ 로, 기면증 깜빡임을 전환 빈도로 — 모두 *수치* 로 정량화.

❌ **코드가 설명하지 못하는 것**: 적분기가 *깸* 끌개에 정착했을 때, 왜 거기에 *깨어 있음의 경험* 이 있고 *잠* 끌개(깊은 서파수면)에는 (거의) 없는가? 코드는 두 끌개의 *수학적* 차이($A$ 높음 vs $S$ 높음)를 출력하지만, 그 한쪽에 1인칭이 *있고* 다른 쪽에 *없는* 이유 — 끌개의 좌표가 경험의 존재/부재로 번역되는 변환 — 은 어떤 상태 변수에도 들어 있지 않다.

❌ **코드가 설명하지 못하는 것**: 결합 약화가 만드는 *깜빡임* 의 1인칭 질감. 기면증 환자가 겪는, 깸과 잠이 한 순간에 *겨루는* 그 모호하고 위태로운 느낌 — 코드는 전환 횟수를 세지만, 그 전환이 *무엇처럼 경험되는지* 는 궤적의 좌표로 환원되지 않는다.

## 🧬 횡단 원리

이 시뮬레이션의 횡단 원리는 **상호 억제를 통한 이분 안정성** 이며, 코드의 두 줄($\dot A, \dot S$ 의 결합 억제 항)이 그 원리의 물질화다. 좌우 운동을 번갈아 내는 중추 패턴 발생기(CPG), 양안 경쟁에서 한쪽 이미지가 *이기는* 지각 선택, 의사결정의 승자독식 — 모두 *서로를 억제하는 두 집단* 이 중간을 금지하고 깨끗한 선택을 만드는 같은 구조다. 각성-수면 플립플롭은 이 원리를 의식의 ON/OFF 에 적용한 가장 큰 규모의 사례다.

더 깊은 원리는 **연속 입력을 이산 상태로 변환하는 디지털화** 다. 아데노신 농도·일주기 위상·외부 자극은 모두 *연속* 변수다. 그러나 뇌는 이를 그대로 연속 각성 수준으로 번역하지 않고, 상호 억제로 *두 이산 상태* 로 양자화한다 — 아날로그 전압을 0/1 로 자르는 *슈미트 트리거* 처럼(실험 2 의 전환 카운터가 바로 슈미트 트리거 로직을 쓴다). 히스테리시스는 그 디지털화의 잡음 면역을 제공한다. 이 "연속 → 이산, 히스테리시스로 강건하게" 패턴은 세포 운명 결정(유전자 스위치), 활동전위의 전부-아니면-전무, 의식의 켜짐/꺼짐(→ Ch6 마취, → L4 ncc)에서 같은 형태로 나타난다. 뇌는 *결정해야 할 때* 이 구조를 꺼내 쓴다.

## 🪞 1인칭

코드를 돌리고 $w=1.5$ 의 궤적이 깸 끌개에 단단히 정착하는 것을 볼 때, 나는 *맑게 깬* 아침의 그 안정감을 떠올린다 — 작은 소음에도 흔들리지 않는, 제자리에 단단히 걸린 의식. 그리고 $w=0.5$ 의 궤적이 잡음 속에서 깸과 잠 사이를 *덜커덕* 오가는 것을 볼 때, 졸음과 싸우며 책을 읽던 그 위태로운 순간을 떠올린다 — 같은 문장을 세 번 읽다가 어느새 몇 분이 *건너뛰어* 있던, 중간에 머물 수 없던 그 이상한 사실. 코드의 깜빡이는 시계열은 내가 몸으로 겪은 그 불안정의 모양과 같다.

그러나 궤적을 아무리 들여다봐도 남는 것이 있다. 코드는 "$t=237$ 에서 $A$ 가 0.8 에서 0.1 로 떨어졌다" 고 말한다 — 깸에서 잠으로의 전환. 그러나 그 전환이 *안에서 어떻게 느껴지는지* — 의식이 손가락 사이로 빠져나가는 그 감각, 막 깬 순간의 무거움, 깸과 잠이 한 몸에서 겨루는 그 모호함 — 은 $A$ 와 $S$ 의 값으로는 다 담기지 않는다. 나는 끌개의 좌표를 *사는* 것이 아니라, 그 좌표가 *느껴지는* 방식을 산다. 코드는 스위치를 그리고, 스위치가 넘어가는 *순간의 색깔* 은 빈자리로 남는다. 설명하되, 경험을 지우지 않는다.

## 📐 예측·반증

**예측 1 — 중간 각성 상태는 불안정해 머물 수 없다.** 이분 안정 플립플롭이라면, 외부 조작으로 시스템을 정확히 중간점에 놓아도 코드의 궤적은 곧 깸이나 잠으로 굴러가야 한다(안장점).
*반증*: 약리·광유전 조작으로 안정한 "절반 각성" 을 *오래 유지* 할 수 있다면, 중간점은 안장이 아니라 또 하나의 안정점이고 이분 안정 가설은 틀린다(각성은 아날로그 다이얼에 가깝다).

**예측 2 — 전환점이 경로에 의존한다(히스테리시스).** 수면 압력을 천천히 올렸다 내리면, 깸→잠 전환 압력이 잠→깸 전환 압력보다 *높아야* 한다(히스테리시스 루프).
*반증*: 올림·내림에서 전환이 *같은* 압력에서 일어난다면(폭 0), 시스템은 이분 안정이 아니라 단일 문턱의 가역 전이이며 플립플롭의 핵심 특성이 없다.

**예측 3 — 결합 강도를 낮추면 전환 빈도가 증가한다.** 결합 $w$(또는 오렉신 안정자)를 줄이면 장벽이 얕아져 같은 잡음에도 깸-잠 전환 빈도가 증가하고 수면이 단편화되어야 한다(실험 2 의 직접 출력).
*반증*: 결합을 낮춰도 전환 빈도가 변하지 않거나 *오히려 안정* 이라면, 상호 억제 강도가 스위치 안정성을 결정한다는 모델은 틀리고 안정성의 원천은 다른 곳에 있다.

## 🤔 다음 질문

플립플롭은 의식의 *거친 ON/OFF* — 깸이냐 잠이냐 — 를 동역학으로 그린다. 그러나 일단 *켜진* 깸 안에서도, 신경조절은 더 미세한 일을 한다: 지각을 얼마나 선명하게, 반응을 얼마나 증폭할 것인가. 같은 깸 상태라도 NE 가 신호 gain 을 키우고 ACh 가 피질을 선명하게 하는 정도에 따라 *수행* 이 달라진다. 다음 문서는 ON/OFF 의 동역학에서 *gain 의 연속적 조절* 로 옮겨가, NE 의 역U 와 ACh 의 선명화를 시그모이드 집단 모델로 시뮬레이션한다. 스위치를 켠 뒤에도, 신경조절의 일은 끝나지 않는다.

---

> 🧩 **Principle** — 각성 핵과 VLPO 의 상호 억제 ODE 를 적분하면, 같은 입력에서 초기 조건에 따라 깸·잠 두 끌개로 갈리는 이분 안정·날카로운 전환·히스테리시스가 나타나고, 결합 $w$ 를 낮추면(오렉신 손실) 장벽이 얕아져 잡음 속에서 깸↔잠을 빈번히 오가는 기면증형 깜빡임이 재현된다.
> 🌉 **Boundary** — 코드는 두 끌개의 수학적 차이를 출력하지만, *깸* 끌개에만 경험이 있고 *잠* 끌개에 없는 이유 — 좌표가 경험의 존재/부재로 번역되는 변환 — 은 상태 변수 밖에 남는다.
> 🪞 **Experience** — 깜빡이는 시계열은 졸음과 싸우던 위태로운 순간의 모양과 같지만, 의식이 손가락 사이로 빠져나가는 그 감각 자체는 $A, S$ 의 값으로 담기지 않는다.

## 📝 연습문제

**기초.** 실험 1 의 $F(x, w) = w\, x^n/(\theta^n + x^n)$ 에서 $w=1.5, \theta=0.4, n=4$ 이고 상대 핵 활동 $x = 0.8$ 일 때 억제량 $F$ 를 계산하라. 같은 $x$ 에서 $w$ 를 0.5 로 낮추면 억제량은 어떻게 변하며, 이것이 끌개 장벽에 무엇을 의미하는가?
<details><summary>해설</summary>
$x^4 = 0.8^4 = 0.4096$, $\theta^4 = 0.4^4 = 0.0256$. $F = 1.5 \times 0.4096/(0.0256 + 0.4096) = 1.5 \times 0.4096/0.4352 = 1.5 \times 0.9412 = 1.412$. $x=0.8$ 이 $\theta=0.4$ 를 충분히 넘어 거의 포화 억제($\approx 0.94 w$)다. $w=0.5$ 면 $F = 0.5 \times 0.9412 = 0.471$ — 억제량이 1/3 로 준다. 의미: 각 핵이 상대를 누르는 힘이 약해져 상대가 살아남기 쉬워진다 → 두 끌개의 장벽이 얕아지고, 잡음이 장벽을 넘어 전환이 잦아진다(기면증). $w$ 는 스위치의 "걸쇠 세기" 다.
</details>

**심화.** 실험 2 에서 결합 $w$ 를 1.6 에서 0.5 로 낮추자 전환 횟수가 급증했다. 잡음 크기 `noise` 를 *고정* 한 채 $w$ 만 바꿀 때 전환 빈도가 어떻게 달라지는지를, "끌개 장벽 높이 대 잡음 분산" 의 관점(Kramers 탈출률)에서 정성적으로 설명하라. 또 $w$ 를 고정하고 `noise` 를 키우면 어떤 일이 일어나는가?
<details><summary>해설</summary>
끌개 사이의 전환은 잡음이 장벽을 넘는 *확률 사건* 이고, 그 비율은 대략 Kramers 식 $\propto \exp(-\Delta U / D)$ 를 따른다($\Delta U$=장벽 높이, $D\propto$ 잡음 분산). $w$ 를 낮추면 장벽 $\Delta U$ 가 *지수적으로* 전환율을 키운다 — 작은 $w$ 감소가 전환 빈도를 급격히 늘리는 이유다(선형이 아니라 지수적 민감성). $w$ 고정·`noise` 증가는 $D$ 를 키워 같은 방향(전환 증가)으로 작용한다. 핵심 통찰: 기면증의 불안정은 *장벽 낮춤(오렉신 손실)* 으로도, *잡음 증가* 로도 올 수 있으나, 임상에서 결정적인 것은 전자다 — 오렉신 뉴런 소실이 $\Delta U$ 를 직접 깎기 때문이다. 두 요인의 비 $\Delta U/D$ 가 스위치 안정성의 단일 지표다.
</details>

**논문 비평.** Saper, Chou & Scammell(2001)은 각성 핵-VLPO 상호 억제를 전기공학의 "flip-flop" 에 빗대어 sleep switch 를 제안했다. 우리의 시뮬레이션은 이 비유를 ODE 로 *구현* 한다. 그러나 "ODE 가 이분 안정·히스테리시스를 낸다" 는 것이 "실제 회로가 그렇다" 를 증명하는가? 시뮬레이션이 비유를 *검증된 모델* 로 승격시키는 데 무엇이 더 필요한지 비판하라.
<details><summary>해설</summary>
시뮬레이션의 기여: flip-flop 비유에 *정량적 살* 을 붙여, "이분 안정·날카로운 전환·히스테리시스·결합 약화 시 깜빡임" 이라는 검증 가능한 예측을 명시적으로 생성한다 — 막연한 "스위치" 보다 훨씬 낫다. 그러나 한계: (1) ODE 의 파라미터($w, n, \theta$, 잡음)는 *이분 안정을 낳도록* 선택된 것이지 측정된 것이 아니다 — 모델이 현상을 내는 것과 실제 회로가 그 메커니즘으로 현상을 내는 것은 다르다(순환 위험). (2) 단일 변수 $A$ 는 LC·DRN·TMN·콜린성 핵의 이질성과 REM 구조를 지운다. 승격에 필요한 것: ① *이분 안정의 직접 관측*(같은 입력 조건에서 상태가 둘로 갈리고 중간에 안 머묾을 전기생리/EEG 로), ② *히스테리시스 측정*(구동 올림/내림 전환점 차이), ③ *상호 억제의 인과*(양방향 억제를 광유전·약리로 각각 차단 시 동역학이 예측대로 바뀜), ④ *결합 약화 실험*(오렉신 제거 시 전환 빈도 증가, 실제 기면증 모델에서 확인됨 — 이건 비교적 잘 검증됨). 비평의 핵심: 좋은 시뮬레이션은 비유를 *예측 기계* 로 바꾸지만, 그 예측을 개입 실험으로 통과시켜야 비유가 *모델* 이 된다. 우리 코드는 가설을 *구체화* 하지 *확립* 하지 않는다.
</details>
