# 🧠 보상예측오차 수학 — δ = r + γV(s′) − V(s)

*슐츠가 원숭이에서 본 도파민 신호는 강화학습의 시간차 오차 $\delta_t = r_t + \gamma V(s_{t+1}) - V(s_t)$ 와 한 치도 어긋나지 않는다 — 신경과학과 인공지능이 같은 방정식을 공유하는 지점.*

## 🎯 핵심 질문

앞 문서에서 우리는 도파민이 "실제 보상 − 예측된 보상" 을 신호한다는 것을 *질적으로* 보았다. 그러나 이 직관을 정밀한 알고리즘으로 바꾸려면 두 가지 질문에 답해야 한다.

첫째, **"예측" 은 정확히 무엇인가?** 단지 다음 한 스텝의 보상에 대한 예측인가, 아니면 미래 전체에 걸친 보상의 합에 대한 예측인가? 둘째, **미래의 보상은 현재 어떻게 평가되는가?** 지금의 1ml 와 10초 뒤의 1ml 는 같은 가치인가?

강화학습의 **시간차(temporal difference, TD) 학습** 은 이 두 질문에 동시에 답한다. 가치 함수 $V(s)$ 를 "상태 $s$ 에서 시작해 미래에 받을 *할인된 보상의 총합* 에 대한 기댓값" 으로 정의하고, 매 스텝의 예측 오차를

$$\delta_t = r_t + \gamma V(s_{t+1}) - V(s_t)$$

로 계산한다. 핵심 질문은 이것이다. **이 한 줄의 방정식이 슐츠의 세 조건 — 예측 초과의 버스트, 예측된 보상의 침묵, 생략의 dip, 그리고 US→CS 신호 이동 — 을 어떻게 *모두* 자동으로 생성하는가?** 이 문서는 그 정확한 대응을 손으로 유도하고 시뮬레이션으로 확인한다.

## 🌍 어디서 마주치나

TD 오차는 추상적 수식이 아니라, 학습이 일어나는 모든 곳에서 작동하는 알고리즘이다.

- **인공지능의 핵심 엔진.** 알파고·아타리 게임 에이전트·로봇 제어 — 현대 강화학습의 가치 추정은 거의 모두 TD 학습의 변형이다. 슐츠의 원숭이가 발견된 1997년과 거의 같은 시기에 Sutton & Barto 가 TD 를 정식화한 것은 우연이 아니다.
- **습관 형성.** 매일 같은 카페에서 커피를 사면, 처음엔 보상이었던 커피가 점점 "당연한" 일상이 되고(RPE→0), 카페 *간판이 보이는 순간* 으로 기대가 이동한다(US→CS).
- **시간 할인.** $\gamma$(감마)는 우리가 미래 보상을 얼마나 평가절하하는지를 정한다. 즉각적 만족을 미래의 더 큰 보상보다 선호하는 충동성은 낮은 $\gamma$ 로 모델링된다.
- **TV 광고의 음악.** 광고 시작 전 익숙한 징글이 흐르면, 그 자체가 보상(제품)을 예측하는 CS 가 되어 가치를 거슬러 운반한다.

## 🔍 직관의 함정

**함정 1: "RPE 는 단지 한 스텝 앞의 보상 예측 오차다."**
TD 오차의 핵심은 $V(s)$ 가 *다음 보상* 이 아니라 *미래 전체의 할인된 보상 합* 을 예측한다는 데 있다. $\delta_t$ 의 부트스트랩 항 $\gamma V(s_{t+1})$ 은 "다음 상태의 가치" 를 끌어와 *부분적 목표* 로 삼는다. 즉 RPE 는 "보상에 대한 오차" 가 아니라 "*가치*(미래 보상 총합)에 대한 오차" 다. 이 차이가 신호의 시간적 이동을 가능하게 한다. 반박됨.

**함정 2: "$\gamma$ 는 그냥 임의의 튜닝 상수다."**
$\gamma \in [0,1)$ 은 단순한 하이퍼파라미터가 아니라 *시간 지평* 을 정의하는 의미 있는 양이다. $\gamma$ 가 0 에 가까우면 근시안적(즉각 보상만), 1 에 가까우면 장기적(먼 미래까지). 또한 $\gamma$ 는 수학적으로 무한 합 $\sum_t \gamma^t r_t$ 의 *수렴* 을 보장한다 — $\gamma=1$ 이면 합이 발산할 수 있다. 신경 수준에서 $\gamma$ 는 세로토닌·할인율과 연결된다는 가설도 있다. 반박됨.

**함정 3: "TD 학습은 보상이 주어진 후에야 학습한다."**
부트스트랩의 마법이 바로 이것이다. TD 는 *최종 보상을 기다리지 않고* 매 스텝마다 학습한다. $V(s_t)$ 를 $r_t + \gamma V(s_{t+1})$ 라는 더 나은 추정치로 갱신하므로, 보상 정보가 *아직 도착하지 않아도* 가치가 한 스텝씩 거슬러 전파된다. 이것이 몬테카를로(전체 에피소드 후 학습)와 결정적으로 다른 점이며, 슐츠의 신호 이동을 *시행 내* 가 아니라 *시행 간* 점진적 후퇴로 설명한다. 반박됨.

## ⚙️ 회로·분자 분석

**분석 1 — 가치 함수와 부트스트랩.** 상태 가치를 미래 할인 보상 합의 기댓값으로 정의한다:
$$V(s_t) = \mathbb{E}\!\left[\sum_{k=0}^{\infty} \gamma^k\, r_{t+k}\right].$$
이 정의로부터 *벨만 방정식* 이 따라온다: $V(s_t) = \mathbb{E}[r_t + \gamma V(s_{t+1})]$. 만약 우리의 추정 $V$ 가 완벽하다면 좌변과 우변이 같아야 하므로, 그 차이 $\delta_t = r_t + \gamma V(s_{t+1}) - V(s_t)$ 는 *예측이 얼마나 틀렸는가* 를 정확히 측정한다. $\delta_t$ 가 0 이면 예측이 자기 일관적이고, 0 이 아니면 갱신이 필요하다. 도파민 위상성 발화가 바로 이 $\delta_t$ 다. $\square$

**분석 2 — TD(0) 갱신 규칙.** 오차를 줄이기 위해 가치를 오차 방향으로 조금씩 민다:
$$V(s_t) \leftarrow V(s_t) + \alpha\, \delta_t, \qquad \alpha \in (0,1].$$
$\alpha$ 는 학습률. $\delta_t > 0$(예측 초과)이면 $V(s_t)$ 를 올리고, $\delta_t < 0$(예측 미달)이면 내린다. 이 단순 규칙을 반복하면 $V$ 가 참 가치로 수렴한다. 신경 수준에서 이 갱신은 도파민이 코르티코-선조체 시냅스의 가소성(LTP/LTD)을 *부호 있게* 조절하는 것으로 구현된다 — 양의 도파민은 강화, 음의 도파민(dip)은 약화. $\square$

**분석 3 — 슐츠 세 조건의 자동 생성.** 단순 CS→(지연)→US 시나리오를 보자. (a) *학습 전*: $V$ 는 어디서나 0 이므로 US 스텝에서 $\delta = r + \gamma\cdot0 - 0 = +1$ (버스트). (b) *학습 후*: 가치가 거슬러 전파되어 US 직전 상태들이 높은 $V$ 를 갖는다. US 스텝에서 $\delta = r + \gamma V_{\text{end}} - V_{\text{pre-US}} \approx 0$ (침묵), 반면 CS 스텝에서 $V(s_{\text{CS}}) > V(s_{\text{pre-CS}})$ 이므로 $\delta = 0 + \gamma V_{\text{CS}} - V_{\text{pre-CS}} > 0$ (CS 버스트). (c) *생략*: US 스텝에서 $r=0$ 이지만 $V_{\text{pre-US}}$ 는 높으므로 $\delta = 0 + 0 - V_{\text{pre-US}} < 0$ (dip). 슐츠의 세 패턴이 *추가 가정 없이* 한 방정식에서 나온다. $\square$

**분석 4 — 적격성 흔적과 TD(λ).** TD(0)은 한 스텝씩만 신용을 할당하므로 신호가 한 시행에 한 칸씩만 후퇴한다. *적격성 흔적(eligibility trace)* $e_t$ 는 최근 방문한 상태들에 *감쇠하는 흔적* 을 남겨, 하나의 오차로 여러 과거 상태를 동시에 갱신한다:
$$e_t(s) = \gamma\lambda\, e_{t-1}(s) + \mathbb{1}[s_t = s], \qquad V(s) \leftarrow V(s) + \alpha\,\delta_t\, e_t(s).$$
$\lambda \in [0,1]$ 은 TD(0)($\lambda=0$)과 몬테카를로($\lambda=1$)를 잇는 다리다. 신경 수준에서 적격성 흔적은 시냅스에 남는 *분자적 표지*(예: CaMKII 활성, 칼슘 농도)로 구현될 수 있어, 도파민이 도착했을 때 "최근 활성화된" 시냅스만 선택적으로 강화되게 한다. 이것이 시간적으로 떨어진 행동과 보상을 잇는 *신용 할당 문제* 의 한 해법이다. $\square$

## 🧪 실험·임상 증거

이론의 원전은 **Montague, Dayan & Sejnowski (1996), "A Framework for Mesencephalic Dopamine Systems Based on Predictive Hebbian Learning", *Journal of Neuroscience* 16:1936** 과 **Schultz, Dayan & Montague (1997), *Science***, 표준 교과서는 **Sutton & Barto (2018), *Reinforcement Learning: An Introduction*** 이다. 아래 실험은 TD(0)을 여러 시행에 걸쳐 돌려 도파민 신호가 US 에서 CS 로 이동하는 과정을 열지도 형태로 출력한다.

### 실험 1 — TD 학습으로 US→CS 신호 이동 재현

```python
import numpy as np

# 슐츠 실험: CS(1번 스텝) → (지연) → US(보상, 마지막 스텝)
# δ_t = r_t + γ V(s_{t+1}) − V(s_t)  를 시행마다 반복해 신호 이동을 본다.
def td_learning(n_trials=60, alpha=0.3, gamma=0.95, n_steps=10, cs_step=1):
    V = np.zeros(n_steps + 1)          # 종단 상태 포함
    da = np.zeros((n_trials, n_steps)) # 시행 × 스텝 도파민(RPE)
    for trial in range(n_trials):
        r = np.zeros(n_steps); r[-1] = 1.0   # US 는 마지막 스텝
        for t in range(n_steps):
            delta = r[t] + gamma * V[t+1] - V[t]   # TD 오차 = 도파민
            V[t] += alpha * delta                  # TD(0) 갱신
            da[trial, t] = delta
    return da, V

da, V = td_learning()
cs, us = 1, 9

print("시행에 따른 도파민(RPE) — CS 스텝 vs US 스텝")
print(f"{'시행':>4} {'CS스텝':>9} {'US스텝':>9}")
for tr in [0, 2, 5, 15, 40, 59]:
    print(f"{tr:>4} {da[tr, cs]:>+9.3f} {da[tr, us]:>+9.3f}")

# 간이 열지도(텍스트): 시행이 진행되며 양의 신호가 US→CS 로 이동
print("\n도파민 신호 열지도 (행=시행, 열=스텝, █강한+ ▓약한+ ·≈0)")
def glyph(x):
    if x > 0.3:  return "█"
    if x > 0.05: return "▓"
    if x < -0.05: return "▽"
    return "·"
for tr in range(0, 60, 6):
    row = "".join(glyph(da[tr, t]) for t in range(10))
    print(f"  t={tr:>2} [{row}]")

print("\n→ 초기엔 US 스텝(우측)이 강하고, 학습되며 CS 스텝(좌측)으로 신호가 이동한다.")
```

실행하면 초기 시행에서는 US 스텝의 RPE 가 `+1.0` 에 가깝고 CS 스텝은 `0` 이지만, 시행이 진행될수록 US 스텝이 `0` 으로 줄고 CS 스텝이 양수로 커진다 — 열지도에서 `█` 이 우측에서 좌측으로 이동하는 것으로 보인다. 슐츠의 신호 이동이 정확히 재현된다.

**반례 노트.** TD 모델은 *연속적이고 동질한* 시간 표상을 가정한다. 그러나 실제 뇌에서 CS-US 간격의 *시간 불확실성* 이 클 때, 생략 dip 의 정확한 시점 예측은 단순 TD(0)과 어긋날 수 있다. 또 도파민 dip 의 깊이가 음의 RPE 크기에 *선형* 으로 비례하지 않는다(바닥 효과, Bayer & Glimcher 2005)는 점은 TD 의 대칭적 부호 가정을 위반한다. TD 는 강력한 1차 근사이지 완전한 생리 모델은 아니다.

## 🌉 설명적 간극

✅ **수학이 설명하는 것**: 신호의 *시간적 이동* 이 왜 일어나는가 — 부트스트랩과 가치 전파의 필연적 귀결. 생략 dip 의 *존재와 시점*. 학습 곡선의 *형태*.

✅ **수학이 설명하는 것**: 시간 할인($\gamma$)과 신용 할당(적격성 흔적)의 정확한 알고리즘. 신경과학과 AI 가 *같은 방정식* 을 공유한다는 통일.

❌ **수학이 설명하지 못하는 것**: $\delta_t$ 라는 스칼라가 *왜 도파민이라는 특정 분자* 로 구현되어야 하는가. 같은 알고리즘은 다른 물질로도 구현될 수 있다 — 방정식은 *구현 독립적* 이다.

❌ **수학이 설명하지 못하는 것**: $\delta_t = +0.7$ 이 *어떻게 느껴지는가*. 방정식은 오차의 크기를 주지만, 그 오차가 동반하는 갈망·만족·실망의 현상적 질은 어떤 변수에도 들어 있지 않다.

## 🧬 횡단 원리

TD 학습의 횡단 원리는 **부트스트랩 — 추정으로 추정을 갱신하라** 다. 뇌는 정답(최종 보상)을 기다리지 않는다. 현재의 불완전한 예측을, *약간 덜 불완전한* 다음 스텝의 예측으로 끌어당겨 한 스텝씩 개선한다. 이것은 베이지안 추론의 순차적 갱신, 칼만 필터의 예측-보정 사이클, 예측 코딩의 계층적 오차 전파와 같은 형식적 가족에 속한다.

더 깊게는, TD 오차가 **자기 일관성의 측정** 이라는 점이다. $\delta_t$ 는 "지금의 예측과 한 스텝 뒤의 예측이 서로 모순되는가" 를 잰다. 모든 예측 시스템은 이런 *내적 모순* 을 0 으로 만들려 한다. 도파민 RPE 는 보상 영역에서 이 자기 일관화 압력이 신경 신호로 물질화된 것이며(→ IQ AI Lab RL 의 TD-error 와 동일), 같은 원리가 감각 피질·운동 학습·언어 예측 전반에 퍼져 있다(→ L6 prediction-everywhere). 슐츠의 원숭이와 알파고는 문자 그대로 같은 $\delta$ 를 계산한다.

## 🪞 1인칭

습관이 형성되는 과정을 안에서 느껴본 적이 있는가. 새로운 출근길에 처음 마주친 멋진 카페 — 그 발견의 기쁨은 강렬했다(US 에서 큰 양의 RPE). 그러나 매일 그 길을 걸으면서, 기쁨은 점점 옅어지고 어느새 그 카페는 풍경의 일부가 되었다(RPE→0). 그리고 더 미묘하게, 이제는 *모퉁이를 도는 순간* 살짝 기분이 좋아진다 — 아직 카페를 보기도 전에. 기대가 보상의 순간에서 그것을 예고하는 단서로 거슬러 올라간 것이다. 당신은 방금 $\gamma V(s_{t+1})$ 의 부트스트랩을 몸으로 경험했다.

그러나 방정식을 아무리 정밀하게 풀어도 남는 것이 있다. TD 는 "모퉁이에서 $\delta = +0.2$" 라고 말할 수 있다. 그러나 그 $+0.2$ 가 *어떤 느낌인지* — 그 작고 따뜻한 기대감, 곧 좋은 것이 온다는 그 어렴풋한 끌림 — 은 어떤 변수에도 적히지 않는다. 우리는 학습의 알고리즘을 완전히 안다. 우리는 학습이 *무엇처럼 느껴지는지* 는 여전히 외부에서만 가리킬 수 있다. 설명하되, 경험을 지우지 않는다.

## 📐 예측·반증

**예측 1.** $\gamma$ 를 낮춘 피험자(또는 높은 시간 할인을 보이는 충동적 개체)에서는, 보상에서 멀리 떨어진 CS 일수록 도파민 신호 이동이 약하거나 더 가까운 단서에만 국한되어야 한다.
*반증*: 시간 할인 행동과 무관하게 모든 CS-US 간격에서 동일한 신호 이동이 관찰되면 $\gamma$ 의 생리적 의미가 약화된다.

**예측 2.** 적격성 흔적($\lambda > 0$)이 작동한다면, 단일 보상 후 *여러* 과거 상태의 가치가 한 시행 만에 동시에 갱신되어야 하며, 학습 속도가 TD(0)보다 빨라야 한다.
*반증*: 가치 전파가 항상 시행당 정확히 한 스텝씩만 후퇴한다면(흔적 없음), TD(λ)의 생리적 구현 가설은 반박된다.

**예측 3.** 보상 예측이 완벽해진 상태에서 CS-US *간격을 갑자기 늘리면*, 원래 예상되던 시점에 음의 dip 이, 새 보상 시점에 양의 버스트가 동시에 나타나야 한다(시간 예측 오차).
*반증*: 간격 변화 시 도파민이 시점 정보 없이 단순히 보상 유무에만 반응한다면, 시간적 RPE 표상이 반박된다.

## 🤔 다음 질문

우리는 이제 도파민이 *무엇을* 계산하는지($\delta_t$) 안다. 그러나 같은 도파민 분자가 뇌의 *어디로* 가느냐에 따라 그 역할이 극적으로 달라진다. VTA 에서 측좌핵으로 가는 신호는 동기를, 전전두피질로 가는 신호는 인지를, SNc 에서 등쪽 선조체로 가는 신호는 운동을 조율한다. 같은 RPE 가 다른 회로에서 다른 일을 한다 — 다음 문서는 도파민의 세 가지 주요 투사 경로를 해부한다.

---

> 🧩 **Principle** — 도파민 위상성 신호는 시간차 오차 $\delta_t = r_t + \gamma V(s_{t+1}) - V(s_t)$ 다. 부트스트랩($\gamma V(s_{t+1})$)과 가치 갱신($V \leftarrow V + \alpha\delta$)이 슐츠의 세 조건과 US→CS 이동을 추가 가정 없이 생성한다.
> 🌉 **Boundary** — 방정식은 구현 독립적이다. *왜 하필 도파민* 인지, $\delta$ 가 *어떻게 느껴지는지* 는 수식 밖에 남는다.
> 🪞 **Experience** — 습관이 굳으며 보상의 기쁨이 옅어지고 기대가 단서로 거슬러 가는 그 미묘한 끌림이, $\gamma V(s_{t+1})$ 의 1인칭 면이다.

## 📝 연습문제

**기초.** $\gamma = 0.9$, 현재 상태 가치 $V(s_t) = 0.5$, 다음 상태 가치 $V(s_{t+1}) = 0.8$, 즉시 보상 $r_t = 0$ 일 때 TD 오차 $\delta_t$ 를 계산하라. 학습률 $\alpha = 0.2$ 로 갱신된 $V(s_t)$ 는?
<details><summary>해설</summary>
$\delta_t = r_t + \gamma V(s_{t+1}) - V(s_t) = 0 + 0.9 \times 0.8 - 0.5 = 0.72 - 0.5 = +0.22$. 양의 오차이므로 가치를 올린다: $V(s_t) \leftarrow 0.5 + 0.2 \times 0.22 = 0.5 + 0.044 = 0.544$. 보상이 없어도 "다음 상태가 예상보다 좋다" 는 정보로 현재 가치가 상승했다 — 부트스트랩의 작동.
</details>

**심화.** CS(스텝 1)→US(스텝 5, 보상 1) 시나리오에서 $\gamma = 1$, $\alpha = 1$ 로 TD(0)을 손으로 돌려라. 첫 시행 후, 둘째 시행 후, 셋째 시행 후 US 직전 상태(스텝 4)의 가치가 어떻게 변하는지 추적하고, 신호가 한 시행에 몇 스텝씩 후퇴하는지 설명하라.
<details><summary>해설</summary>
$\alpha=1, \gamma=1$ 이면 갱신이 즉각적이다. 첫 시행: 스텝 5 에서 보상 1 을 받아 스텝 4 의 가치가 $V_4 = 1$ 로 갱신(스텝 5 직전에서 $\delta = 1 + 0 - 0 = 1$). 그러나 스텝 3 은 아직 $V_3 = 0$(스텝 4 의 가치가 갱신되기 *전에* 스텝 3→4 전이를 평가했으므로). 둘째 시행: 이제 $V_4 = 1$ 이므로 스텝 3→4 에서 $\delta = 0 + 1 - 0 = 1$, $V_3 = 1$ 로 갱신. 셋째 시행: $V_2 = 1$. 즉 TD(0)은 *시행당 한 스텝씩* 가치(그리고 양의 RPE)를 CS 쪽으로 후퇴시킨다. 이것이 슐츠의 점진적 신호 이동의 알고리즘적 기원이며, 적격성 흔적이 있으면 이 후퇴가 한 번에 여러 스텝 진행된다.
</details>

**논문 비평.** Montague, Dayan & Sejnowski (1996)와 Schultz, Dayan & Montague (1997)는 도파민 = TD 오차라는 대응을 확립했다. 이 "도파민-TD 대응" 가설의 *강점* 과, 이 대응이 *과대 해석* 될 위험을 함께 논하라. 특히 알고리즘 수준의 일치가 곧 도파민이 "단지 TD 오차" 임을 함의하는지 비판적으로 검토하라.
<details><summary>해설</summary>
강점: (1) *예측력* — TD 는 슐츠의 세 조건, 신호 이동, 생략 dip 을 단일 방정식에서 자동 생성하며 사후 끼워맞춤이 아니다. (2) *통일성* — 신경과학과 AI 가 같은 $\delta$ 를 공유함을 보여 마(Marr)의 계산 수준에서 강력한 설명을 준다. (3) *정량성* — 부분 강화·확률적 보상에서 정량적 예측을 내고 검증되었다. 과대 해석 위험: (1) *구현 ≠ 알고리즘* — 도파민이 TD 오차와 *상관* 한다고 해서 뇌가 문자 그대로 $V$ 테이블을 갱신한다는 뜻은 아니다(분산 코딩, 다른 표상 가능). (2) *완전성 오류* — 3장의 반례(현저성·램핑·다중 신호)가 보이듯 도파민은 RPE 를 *포함* 하지만 RPE 로 *환원* 되지 않는다. (3) *마의 함정* — 계산 수준의 우아함이 알고리즘·구현 수준의 다양성을 가릴 수 있다. 결론: 도파민-TD 대응은 신경과학 최고의 계산 이론 중 하나지만, "도파민 = TD 오차" 는 "도파민의 핵심 함수 중 하나가 TD 오차" 로 신중히 읽어야 한다.
</details>
