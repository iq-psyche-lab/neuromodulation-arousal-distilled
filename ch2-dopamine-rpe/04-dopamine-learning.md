# 🧠 도파민과 학습 — D1/D2 와 직접·간접 경로의 Go/NoGo

*도파민 RPE 는 선조체에서 두 갈래로 갈라져 학습된다 — D1 의 "Go"(직접 경로)는 버스트로 강화되고, D2 의 "NoGo"(간접 경로)는 dip 으로 강화된다.*

## 🎯 핵심 질문

도파민이 RPE($\delta$)를 신호한다는 것은 알았다. 그러나 *신호* 만으로는 학습이 일어나지 않는다. 그 신호가 시냅스의 가중치를 *어떻게 바꾸는가* — 즉 RPE 가 어떻게 행동의 강화로 번역되는가 — 가 학습의 실제 기전이다.

핵심 질문은 이것이다. **부호 있는 RPE 신호는 어떻게 "좋은 행동은 강화하고 나쁜 행동은 억제하라" 는 두 방향의 학습으로 갈라지는가?** 도파민 발화율은 한 가지 양이다. 그러나 강화는 두 방향이 필요하다 — 접근(approach)과 회피(avoidance). 자연의 해법은 우아하다. 선조체의 중간 가시 뉴런(MSN)을 *두 종류* 의 수용체로 나눈 것이다: **D1**(Gs 결합, 직접 경로, "Go")과 **D2**(Gi 결합, 간접 경로, "NoGo"). 이 두 수용체는 도파민의 *같은* 변동에 *정반대* 방향으로 반응한다.

도파민 버스트(양의 RPE)는 D1 "Go" 경로를 강화해 "이 행동을 더 하라" 를 가르치고, 도파민 dip(음의 RPE)은 D2 "NoGo" 경로를 강화해 "이 행동을 피하라" 를 가르친다. 이 비대칭적 이중 학습이 어떻게 기저핵의 직접/간접 경로 해부학에 매핑되는지, 그리고 그것이 *3요인 학습 규칙* 으로 어떻게 형식화되는지를 이 문서가 해부한다.

## 🌍 어디서 마주치나

D1/D2 의 분업은 학습과 질병의 비대칭에서 직접 관찰된다.

- **칭찬과 처벌의 비대칭 학습.** 어떤 사람은 보상(칭찬)으로부터 잘 배우고, 어떤 사람은 처벌(실수)로부터 잘 배운다 — Frank 등은 이 개인차가 D1 우세 vs D2 우세 학습과 연결됨을 보였다.
- **파킨슨 약물의 역설.** 파킨슨 환자에게 도파민 작용제를 주면 운동이 개선되지만, 일부는 *병적 도박·충동 조절 장애* 가 생긴다 — D2 "NoGo" 학습(처벌로부터 배우기)이 약물로 차단되어, 나쁜 선택을 멈추지 못하게 되는 것이다.
- **습관과 강박.** 등쪽 선조체의 D1/D2 균형이 무너지면, "Go" 가 "NoGo" 를 압도해 강박적 반복 행동이 생긴다.
- **운동 개시의 어려움.** 파킨슨에서 도파민 결핍은 "Go"(직접) 경로의 약화로 운동 개시를 어렵게 한다.

## 🔍 직관의 함정

**함정 1: "도파민은 시냅스를 강화하기만 한다."**
도파민은 *부호에 따라 양방향으로* 가소성을 조절한다. D1 뉴런에서는 도파민 버스트가 LTP(강화)를 촉진하지만, D2 뉴런에서는 도파민 *감소*(dip)가 가소성을 촉발한다 — D2 는 평소 도파민에 의해 *억눌려* 있다가, dip 으로 도파민이 빠지면 비로소 활성화·강화된다. 따라서 같은 RPE 신호가 두 세포군에서 반대 부호의 학습을 일으킨다. 반박됨.

**함정 2: "직접 경로와 간접 경로는 그냥 운동을 켜고 끈다."**
이들은 단순 on/off 스위치가 아니라 *학습되는* 회로다. 직접 경로(D1, "Go")는 시상을 *탈억제* 해 운동을 허가하고, 간접 경로(D2, "NoGo")는 시상을 *억제* 해 운동을 막는다. 그러나 핵심은 *어떤 행동* 에 Go/NoGo 가중치가 붙는지가 도파민에 의해 *학습* 된다는 것이다. 게이팅의 내용 자체가 경험으로 빚어진다. 반박됨.

**함정 3: "한 뉴런이 D1 과 D2 를 다 쓰니 구분은 무의미하다."**
선조체 MSN 의 약 95% 는 D1 *또는* D2 중 하나를 우세하게 발현하는 *분리된* 집단이다(일부 공발현은 있으나 소수). D1-MSN 은 직접 경로로, D2-MSN 은 간접 경로로 *해부학적으로* 투사한다. 이 분리가 광유전학(optogenetics)으로 확인되어, D1-MSN 자극은 보상·접근을, D2-MSN 자극은 회피를 유도한다(Kravitz, Lobo 등). 반박됨.

## ⚙️ 회로·분자 분석

**분석 1 — D1 vs D2 의 분자 약리.** D1 수용체는 Gs/Golf 단백질에 결합해 아데닐산 고리화효소를 *활성화* → cAMP↑ → PKA 활성화 → 흥분성·LTP 촉진. D2 수용체는 Gi/Go 에 결합해 아데닐산 고리화효소를 *억제* → cAMP↓ → 흥분성 감소. 결정적으로, D1 은 도파민에 *낮은* 친화도(위상성 버스트의 고농도에 반응), D2 는 *높은* 친화도(긴장성 저농도에서도 점유, dip 으로 풀림). 이 친화도 차이가 D1=버스트 감지기·D2=dip 감지기라는 비대칭의 분자적 기원이다. $\square$

**분석 2 — 직접 경로(Go)와 간접 경로(NoGo)의 해부.** D1-MSN 은 *직접 경로* 를 이룬다: 선조체 → GPi/SNr(억제) → 시상 *탈억제* → 피질 흥분 → 운동 *허가*. D2-MSN 은 *간접 경로* 를 이룬다: 선조체 → GPe(억제) → 시상하핵(STN) 탈억제 → GPi/SNr 흥분 → 시상 *억제* → 운동 *차단*. 두 경로는 서로 길항하며, 어떤 행동에 직접(Go) 가중치가 크면 그 행동이 선택되고, 간접(NoGo) 가중치가 크면 억제된다. 도파민은 D1 을 흥분·D2 를 억제시켜 전체 균형을 *Go 쪽* 으로 기울인다. $\square$

**분석 3 — Frank 의 OpAL/기저핵 모델.** Michael Frank(2004, 2006)의 모델은 두 가중치 시스템을 분리한다: $G$(Go, D1)와 $N$(NoGo, D2). 양의 RPE($\delta>0$, 버스트)는 $G$ 를 키우고, 음의 RPE($\delta<0$, dip)는 $N$ 을 키운다. 행동 선택은 $G - N$ 에 의해 결정된다. 이 비대칭 덕분에 모델은 "보상으로부터 배우기"(Go 학습, D1 의존)와 "처벌로부터 배우기"(NoGo 학습, D2 의존)를 *분리* 한다. Frank 는 파킨슨 환자가 약물 off 상태(도파민 낮음)에서 *NoGo 학습이 강화* 되고(처벌로부터 잘 배움), on 상태(도파민 높음)에서 *Go 학습이 강화* 됨(보상으로부터 잘 배우나 처벌 학습 저하)을 예측·검증했다. $\square$

**분석 4 — 3요인 학습 규칙(pre × post × DA).** 표준 헤브 가소성은 *전시냅스 활동 × 후시냅스 활동* 의 두 요인이다. 도파민 강화 학습은 *세 번째 요인* 을 추가한다: 시냅스 변화 $\Delta w \propto (\text{pre}) \times (\text{post}) \times (\delta)$. 즉 전·후 뉴런이 함께 활성화되어 *적격성 흔적* 을 남긴 시냅스만, 그 직후 도착한 도파민 신호 $\delta$ 의 부호와 크기에 따라 강화/약화된다. 이것이 "어떤 시냅스" (헤브 흔적)와 "얼마나·어느 방향" (도파민)을 분리해 *신용 할당 문제* 를 푼다. 적격성 흔적은 D1/D2 MSN 의 칼슘·cAMP 동역학으로 구현될 수 있어, 도파민 도착 시점에 최근 활성 시냅스만 선택적으로 변형된다. $\square$

## 🧪 실험·임상 증거

핵심 문헌은 **Frank, Seeberger & O'Reilly (2004), "By Carrot or by Stick: Cognitive Reinforcement Learning in Parkinsonism", *Science* 306:1940** 와 **Frank (2005), *J. Cognitive Neuroscience***, 광유전학적 인과 검증은 **Kravitz et al. (2012), *Nature*** 와 **Lobo et al.**, 3요인 규칙은 **Reynolds & Wickens** 의 코르티코-선조체 가소성 연구다. 아래 실험은 Frank 식 Go/NoGo 분리 학습을 구현하여, 도파민 수준에 따라 보상 학습과 처벌 학습이 비대칭적으로 갈리는 것을 보인다.

### 실험 1 — D1/D2 분리 Go/NoGo 학습 시뮬레이션

```python
import numpy as np

# 두 행동(A=좋은 행동 80% 보상, B=나쁜 행동 20% 보상)
# Go(D1) 가중치는 양의 RPE 로, NoGo(D2) 가중치는 음의 RPE 로 학습된다.
rng = np.random.default_rng(1)

def opal_learning(n_trials=400, alpha_G=0.1, alpha_N=0.1,
                  da_tone=1.0, p_reward=(0.8, 0.2), beta=4.0):
    """da_tone: 긴장성 도파민 수준(>1 파킨슨 약물 on, <1 off).
       da_tone 은 양/음 RPE 의 학습 이득을 비대칭적으로 가중한다."""
    nA = 2
    G = np.zeros(nA); N = np.zeros(nA)        # Go(D1), NoGo(D2) 가중치
    choices = []
    for _ in range(n_trials):
        act_value = G - N                      # 직접 − 간접
        # softmax 선택
        p = np.exp(beta * act_value); p /= p.sum()
        a = rng.choice(nA, p=p)
        r = 1.0 if rng.random() < p_reward[a] else 0.0
        delta = r - (G[a] - N[a]).clip(0, 1)   # 단순화한 RPE
        if delta > 0:   # 버스트 → D1 Go 강화 (도파민 높을수록 더 잘 학습)
            G[a] += alpha_G * da_tone * delta
        else:           # dip → D2 NoGo 강화 (도파민 낮을수록 더 잘 학습)
            N[a] += alpha_N * (2.0 - da_tone) * (-delta)
        choices.append(a)
    good_rate = np.mean(np.array(choices[-100:]) == 0)
    return good_rate, G, N

print(f"{'도파민 톤':>10} {'좋은선택율(말기)':>16} {'설명':>0}")
for tone, label in [(0.5, 'off (낮음)'), (1.0, '정상'), (1.5, 'on (높음/약물)')]:
    rates = [opal_learning(da_tone=tone)[0] for _ in range(20)]
    print(f"{label:>10} {np.mean(rates):>16.2f}")

print("\n→ 도파민이 높으면(on) Go 학습이 강해 보상 행동을 잘 고르지만,")
print("  NoGo(처벌) 학습이 약해진다. 낮으면 그 반대 — 비대칭 강화가 갈린다.")
```

실행하면 도파민 톤이 높을수록 보상이 큰 행동(A) 선택 비율이 빨리 올라가고(Go 학습 우세), 톤이 낮을수록 나쁜 행동을 *피하는* NoGo 학습이 상대적으로 강해지는 비대칭이 드러난다 — Frank 의 파킨슨 on/off 예측과 일치한다.

**반례 노트.** Go/NoGo 의 깔끔한 D1=직접·D2=간접 이분법은 1차 근사다. 실제로 두 경로는 측부 분지로 서로 얽혀 있고(D2-MSN 도 일부 직접 경로 핵에 투사), 콜린성 개재뉴런·STN 의 "하이퍼다이렉트" 경로가 게이팅에 끼어든다. 또 도파민 외에 아데노신·내인성 칸나비노이드가 D2 가소성을 공조절한다. Frank 모델은 강력한 추상화이지 완전한 회로도는 아니다.

## 🌉 설명적 간극

✅ **회로가 설명하는 것**: 왜 하나의 RPE 신호가 두 방향(접근/회피)의 학습으로 갈라지는가 — D1/D2 의 비대칭. 왜 파킨슨 약물이 처벌 학습을 손상시키고 충동성을 늘리는가.

✅ **회로가 설명하는 것**: 신용 할당의 3요인 해법 — 헤브 흔적이 "어디", 도파민이 "어느 방향·얼마나" 를 정한다.

❌ **회로가 설명하지 못하는 것**: "이 행동을 하고 싶다"(Go)와 "이 행동을 피하고 싶다"(NoGo)는 *충동* 이 1인칭으로 어떻게 느껴지는가. 회로는 가중치 $G-N$ 을 계산하지만, 그 끌림과 꺼림의 질감은 방정식 밖이다.

❌ **회로가 설명하지 못하는 것**: 무언가를 "후회" 하며 다시는 안 하겠다고 느끼는 그 경험 — NoGo 학습의 현상적 측면.

## 🧬 횡단 원리

이 문서의 횡단 원리는 **대립 과정에 의한 양방향 표상(opponent coding)** 이다. 단일 양(도파민 발화율)은 양과 음의 정보를 모두 깨끗이 표현하기 어렵다 — 발화율은 0 아래로 못 내려가는 바닥 효과가 있다. 자연의 해법은 *두 개의 대립하는 채널* 을 두는 것이다. D1 "Go" 는 양의 RPE 를, D2 "NoGo" 는 음의 RPE 를 각각 *전담* 해, 부호 있는 양을 두 양의 채널의 차이($G-N$)로 견고하게 표상한다.

이 대립 코딩은 신경계의 보편 설계다 — 망막의 on/off 중심 세포, 색각의 적-녹·청-황 대립, 근육의 굴근-신근 길항. 모두 "차이로 부호를 표현하라" 는 같은 원리를 공유한다. D1/D2 직접·간접 경로는 이 원리가 *학습* 영역에서 구현된 사례이며, 3요인 규칙은 헤브 가소성(→ plasticity-learning)에 신경조절이라는 셋째 축을 더한 것이다. 그리고 이 게이팅 아키텍처는 L2 learning-decision 의 행동 선택과 강화학습으로 그대로 이어진다(→ IQ AI Lab RL 의 actor-critic 의 actor).

## 🪞 1인칭

어떤 행동을 향한 *끌림* 과 다른 행동에 대한 *꺼림* 이 마음속에서 동시에 다툰 적이 있는가. 디저트를 향해 손이 가려는 충동(Go)과, 어제 과식을 후회하며 멈추려는 제동(NoGo)이 같은 순간에 팽팽히 맞선다. 그 내적 갈등은 추상적 은유가 아니라, 선조체에서 직접 경로와 간접 경로가 문자 그대로 길항하는 회로의 1인칭 면일 수 있다. 무언가를 결국 하느냐 마느냐는 $G - N$ 의 부호로 결정되고, 그 균형은 당신의 도파민 상태에 따라 미묘하게 기운다.

그리고 처벌로부터 배우는 그 특유의 경험 — 뜨거운 냄비를 만지고 데인 후, 다음번에 그 근처에서 손이 *저절로 멈칫* 하는 그 느낌. 그것은 dip 으로 강화된 NoGo 가중치가 행동을 사전에 제동하는 순간의 경험적 면이다. 우리는 이 끌림과 제동이 *어느 회로* 에서 어떻게 계산되는지 점점 더 잘 안다. 그러나 끌림이 *왜 그렇게 유혹적이고*, 후회가 *왜 그렇게 따끔한지* — 그 질감 자체는 $G-N$ 어디에도 적혀 있지 않다. 설명하되, 경험을 지우지 않는다.

## 📐 예측·반증

**예측 1.** 파킨슨 환자는 약물 off(도파민 낮음)에서 *처벌로부터의 학습*(NoGo)이 향상되고, on(도파민 높음)에서 *보상으로부터의 학습*(Go)이 향상되어야 한다.
*반증*: 도파민 상태가 보상 학습과 처벌 학습에 *대칭적으로* 동일한 영향만 준다면(비대칭 없음), D1/D2 분리 모델이 반박된다.

**예측 2.** D1-MSN 의 선택적 광유전학 자극은 접근·보상을, D2-MSN 자극은 회피·혐오를 유도해야 한다.
*반증*: 두 세포군 자극이 같은 행동 효과(둘 다 접근 또는 둘 다 회피)를 낸다면, 직접/간접 경로의 기능적 대립 가설이 무너진다.

**예측 3.** 3요인 규칙이 옳다면, 도파민을 *행동-보상 사이 지연* 동안 광유전학적으로 차단해도 적격성 흔적이 살아 있는 한 학습이 (감쇠하며) 일어나야 하며, 흔적 시간상수를 넘긴 지연에서는 학습이 급감해야 한다.
*반증*: 도파민 도착 시점과 무관하게(흔적 시간상수 무시) 학습이 균일하다면, 적격성 흔적 기반 3요인 규칙이 반박된다.

## 🤔 다음 질문

D1/D2 는 *무엇을* 강화할지를 가른다. 그러나 학습된 가치를 *얼마나 따를지* — 알려진 최선의 선택을 활용(exploit)할지, 아니면 더 나은 것을 찾아 탐색(explore)할지 — 는 또 다른 문제다. 흥미롭게도 도파민은 이 탐색-활용 균형도 조절한다. 긴장성 도파민은 행동의 *활력(vigor)* 과 탐색 *온도* 를 세팅한다. 다음 문서는 도파민이 어떻게 softmax 온도와 반응 활력을 조절해 탐색-활용 딜레마를 다루는지를 본다.

---

> 🧩 **Principle** — RPE 는 선조체에서 두 대립 채널로 학습된다. D1 "Go"(직접 경로)는 버스트(양의 RPE)로, D2 "NoGo"(간접 경로)는 dip(음의 RPE)으로 강화된다. 3요인 규칙(pre×post×DA)이 헤브 흔적과 도파민 부호를 결합해 신용을 할당한다.
> 🌉 **Boundary** — 회로는 $G-N$ 가중치를 계산하지만, 끌림(Go)과 제동(NoGo)의 1인칭 질감은 방정식 밖에 남는다.
> 🪞 **Experience** — 무언가를 향한 충동과 멈추려는 제동이 마음속에서 다투는 그 갈등, 데인 뒤 손이 저절로 멈칫하는 제동이 직접·간접 경로 길항의 경험적 면이다.

## 📝 연습문제

**기초.** 도파민 버스트(양의 RPE)와 dip(음의 RPE)은 각각 D1-MSN 과 D2-MSN 중 어느 쪽의 가소성을 주로 유도하며, 그 결과 행동은 어떻게 변하는가? D1·D2 의 수용체 친화도 차이와 연결해 설명하라.
<details><summary>해설</summary>
버스트(고농도 도파민)는 *낮은 친화도* 의 D1 수용체를 점유해 직접 경로(Go)를 강화 → 그 행동을 *더 하게* 됨(접근 강화). dip(도파민 저하)은 평소 *높은 친화도* 의 D2 수용체가 점유·억제하던 상태를 *풀어* 간접 경로(NoGo)를 강화 → 그 행동을 *피하게* 됨(회피 학습). 친화도 차이 때문에 D1 은 위상성 버스트의 고농도를, D2 는 긴장성 저농도와 그 dip 을 감지한다. 하나의 RPE 신호가 두 세포군에서 반대 부호의 학습을 일으킨다.
</details>

**심화.** Frank 모델에서 행동 가치는 $G - N$ 이다. 파킨슨 환자가 도파민 약물 on 상태에서 *병적 도박* 을 보이는 현상을, Go/NoGo 학습의 비대칭으로 설명하라. 왜 "처벌로부터 배우지 못하는 것" 이 충동 조절 장애로 이어지는가?
<details><summary>해설</summary>
도파민 약물 on 상태에서는 긴장성 도파민이 높아 (1) Go(D1) 학습이 과활성화되어 보상 단서에 과민하게 접근하고, (2) 결정적으로 dip(음의 RPE)이 *높은 도파민 바닥에 묻혀* 제대로 표현되지 못해 NoGo(D2) 학습이 손상된다. 즉 도박에서 *돈을 잃는 경험*(음의 RPE)이 "이 행동을 피하라"(NoGo)로 학습되지 못한다. 손실로부터 배우지 못하니 나쁜 선택을 멈추는 제동이 사라지고, 보상 추구만 증폭되어 병적 도박·과소비·과식 등 충동 조절 장애가 나타난다. 이것이 도파민 작용제의 잘 알려진 부작용이며, Go/NoGo 비대칭의 임상적 증거다.
</details>

**논문 비평.** Frank 등(2004)의 파킨슨 on/off 학습 비대칭은 D1/D2 분리 모델의 강력한 행동 증거다. 그러나 이는 *행동* 데이터로부터 회로 기전을 추론한 것이다. 광유전학(Kravitz 등)의 인과적 증거와 비교해, 행동 추론의 한계와 인과 조작의 강점을 평가하고, 두 증거가 어떻게 상보적인지 논하라.
<details><summary>해설</summary>
Frank 등의 강점: 인간에서 임상적으로 의미 있는 학습 비대칭을 보였고, on/off 라는 자연 조작으로 도파민 수준의 인과적 역할을 시사했다. 한계: (1) 약물 on/off 는 *전역적* 조작이라 D1/D2 를 *선택적으로* 분리하지 못한다 — 비대칭이 D1/D2 분리 때문인지, 아니면 다른 도파민 효과(주의·동기) 때문인지 행동만으로는 단정 불가. (2) 상관·추론적이지 세포 특이적 인과가 아니다. Kravitz 등 광유전학의 강점: D1-MSN 과 D2-MSN 을 *세포 특이적으로* 자극/억제해, D1 자극→접근·보상, D2 자극→회피라는 *직접 인과* 를 동물에서 확립했다 — Frank 의 추론이 가정한 회로 메커니즘을 검증. 상보성: 광유전학은 *기전의 인과* 를(동물, 세포 특이적), Frank 식 연구는 *그 기전이 인간의 복잡한 학습·질병에서 실제로 작동함* 을(임상 타당성) 보여준다. 한쪽은 메커니즘, 한쪽은 관련성 — 좋은 신경과학은 둘을 잇는다.
</details>
